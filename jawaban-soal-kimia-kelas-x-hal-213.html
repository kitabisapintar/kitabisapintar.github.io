1. Gunakan tabel untuk menentukan titik impas (break event point) dari persamaan berikut

 P = C (pendapatan = pengeluaran)

  a. C = 15x + 150 dan P = 45x

      x = 0 →  C = 15x + 150 = 15 (0) + 150 = 150

                    P = 45x = 45 (0) = 0

      x = 1 →  C = 15x + 150 = 15 (1) + 150 = 165

                  P = 45x = 45 (1) = 45

      x = 2 → C = 15x + 150 = 15 (2) + 150 = 180

                  P = 45x = 45 (2) = 90

      x = 3 → C = 15x + 150 = 15 (3) + 150 = 195

                  P = 45x = 45 (3) = 135

      x = 4 → C = 15x + 150 = 15 (4) + 150 = 210

                   P = 45x = 45 (4) = 180

      x = 5 → C = 15x + 150 = 15 (5) + 150 = 225

                   P = 45x = 45 (5) = 225

    Titik imbas dari kedua persamaan adalah (5, 225)

b. C = 24x + 80  dan P = 44x

    Titik imbas dari kedua persamaan adalah (4, 176)

c. C = 36x + 200  dan P = 76x

4. Biaya untuk merawat serta perbaikan kuda dan kereta dinyatakan dalam persamaan C = 15.000x + 2.000.000,dimana x adalah banyaknya berkeliling. Jika ongkos untuk satu kali naik berkeliling kota adalah Rp 35.000,00,tentukan:  

Jawab :

a. Persamaan pendapatan yg diperoleh

Pendapatan (P) = 35.000x

Pendapatan bersih = ongkos tiap naik - biaya perawatan (C)

= 35.000 x - (15.000x + 2.000.000)

= 20.000x - 2.000.000

b. Banyak kali kereta untuk berkeliling kota supaya memperoleh break-even point apabila P = C (pendapatan = pengeluaran)

35.000x = 15.000x + 2.000.000

35.000x - 15.000x = 2.000.000

20.000x = 2.000.000

x = 2.000.000 / 20.000

x = 100

Jadi kereta untuk berkeliling kota supaya memperoleh break-even point adalah 100 kali


5. Tugas MTK dan IPA untuk dikerjakan dirumah sebanyak 42 Soal. tugas MTK yg kalian peroleh 10 soal lebih banyak daripada IPA. berapa banyak soal untuk setiap mata pelajaran?  

Misalkan :   x = Matematika

                  y = IPA

x + y = 42 .... pers I

x = 10 + y ⇔ x - y = 10  ... pers II

eliminasi persamaan I dan II

x + y = 42

x - y = 10

------------- +

2x    = 52

   x = 52/2

   x = 26

Subtitusikan

x + y = 42

26 + y = 42

      y = 42 - 26

      y = 16

Jadi banyak soal untuk setiap mata pelajaran adalah matematika = 26 soal dan IPA = 16 soal.
