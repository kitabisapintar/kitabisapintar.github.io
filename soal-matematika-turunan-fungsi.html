Turunan pertama dari f(x) = x² (3x - 1)³ adalah f'(x) = 135x⁴ - 108x³ + 27x² - 2x.

Pembahasan

Soal di atas dapat diselesaikan dengan rumus turunan fungsi aljabar yang penjelasannya sebagai berikut.

Jika f(x) = axⁿ dengan a konstanta real tidak nol dan n bilangan bulat positif, maka: f'(x) = n · axⁿ⁻¹

Jika f(x) = u(x) ± v(x) dengan u(x) dan v(x) masing-masing adalah fungsi yang mempunyai turunan u'(x) dan v'(x), maka: f'(x) = u'(x) ± v'(x)

↓↓↓

Agar soal di atas dapat diselesaikan sebagaimana cara penyelesaian di atas, jabarkan terlebih dahulu soal tersebut.

f(x) = x² (3x - 1)³

f(x) = x² ((3x - 1)² (3x - 1))

f(x) = x² ((9x² - 6x + 1)(3x - 1))

f(x) = x² (27x³ - 27x² + 9x - 1)

f(x) = 27x⁵ - 27x⁴ + 9x³ - x²

Maka turunan dari f(x) = 27x⁵ - 27x⁴ + 9x³ - x² adalah sebagai berikut.

f'(x) = 27 × 5x⁵⁻¹ - 27 × 4x⁴⁻¹ + 9 × 3x³⁻¹ - 2x²⁻¹

f'(x) = 135x⁴ - 108x³ + 27x² - 2x

Jadi turunan pertama dari f(x) = x² (3x - 1)³ adalah f'(x) = 135x⁴ - 108x³ + 27x² - 2x.

12. SOAL MATEMATIKA Tentukan nilai kritis dari fungsi f(x) = 8 ln x - x². Di interval mana fungsi f(x) turun dan di interval mana fungsi f(x) naik?


f(x) = 8.ln(x) - x²

f'(x) = 0 = 8.1/x - 2x

2x = 8/x

2x² = 8

x² = 4

x²-4 = 0

naik ketika f'(x) > 0

x²-4 > 0

(x+2)(x-2) > 0

nilai negatif tidak boleh pada logaritma, jadi x < -2 tidak berlaku

naik pada interval : x > 2

turun ketika f'(x) < 0

x²-4 < 0

(x+2)(x-2) < 0

turun pada interval : x<2 (x > -2 tidak berlaku untuk logaritma), logaritma juga tidak boleh 0 nilai dalamnya, maka interval turunnya jadi 0<x<2

titik kritis : x = 2

Interval dimana sebuah fungsi sedang naik, adalah dimana turunan pertama fungsi lebih besar dari 0. Sebaliknya, fungsi akan turun pada saat interval dimana turunan pertama fungsi lebih kecil dari 0.

Pendahuluan

Turunan atau derivatif adalah bentuk perubahan suatu fungsi setiap nilai dari fungsi tersebut dimasukan, dalam suatu fungsi f(x), misal turunan f(x) adalah f'(x) maka gradien garis singgung f(x) di titik berabsis a adalah f'(a)

Rumus rumus turunan pada fungsi f(x) adalah :

[tex] \sf \frac{df}{dx} (x) = \displaystyle \lim_{\sf h \to 0} \sf \frac{f(x + h) -f(x)}{h} [/tex]

[tex] \sf \frac{d}{dx} (ax^n) = n \cdot ax^{n -1} [/tex]

[tex] \sf \frac{d}{dx} (f(x) + g(x)) = \frac{df}{dx} (x) + \frac{dg}{dx} (x) [/tex]

[tex] \sf \frac{d}{dx} ( C \cdot f(x) ) = C \cdot \frac{df}{dx} (x), C \in konstanta \: acak [/tex]

[tex] \sf misal \: \frac{df}{dx} (x) = f'(x) [/tex]

maka :

(f × g)'(x) = f'(x)g(x) + f(x)g'(x)

[tex] \begin{pmatrix} \sf \frac{f}{g} \end{pmatrix}' \sf (x) = \frac{f'(x) g(x) - f(x) g'(x)}{g^2 (x)} [/tex]

Ada juga rumus aturan rantai :

[tex] \sf \frac{df}{dx} = \frac{df}{du} \times \frac{du}{dx} [/tex]

dan juga bisa menggunakan rantai yang lebih panjang lagi (lebih dari 1 permisalan)

untuk rumus turunan trigonometri, ada rumus umumnya, tetapi untuk bentuk yang lebih rumit lagi bisa diakal akalin lagi dengan rumus turunan yang tadi, rumusnya :

[tex] \sf \frac{d}{dx} ( \sin (x)) = \cos (x) [/tex]

[tex] \sf \frac{d}{dx} ( \cos(x)) = - \sin (x) [/tex]

[tex] \sf \frac{d}{dx} ( \tan(x)) = \sec^2 (x) [/tex]

dengan menggunakan rumus tadi yaitu

[tex] \begin{pmatrix} \sf \frac{f}{g} \end{pmatrix}' \sf (x) = \frac{f'(x) g(x) - f(x) g'(x)}{g^2 (x)} [/tex]

didapat :

[tex] \sf \frac{d}{dx} ( \csc(x)) = - \csc (x) \cot (x) [/tex]

[tex] \sf \frac{d}{dx} ( \sec(x)) = \sec (x) \tan (x) [/tex]

[tex] \sf \frac{d}{dx} ( \cot(x)) = - \csc ^2 (x) [/tex]

Diketahui

f(x) = cot²(2x⁴)

Ditanya

tentukan turunan pertama f(x)!

Penyelesaian

f(x) = cot²(2x⁴)

Gunakan konsep turunan rantai :

[tex] \sf misalkan \: u = 2x^4 \: maka \: \frac{du}{dx} = 8x^3 [/tex]

[tex] \sf misalkan \: v = \cot(2x^4) = \cot(u) \\ \sf maka \: \frac{dv}{du} = - \csc^2 (u) [/tex]

[tex] \sf maka \: f(x) = v^2 \: maka \: \frac{df}{dv} (x) = 2v [/tex]

Dengan menggunakan rumus turunan rantai, didapat :

[tex] \sf \frac{df}{dx} (x) = \frac{df}{dv} (x) \times \frac{dv}{du} \times \frac{du}{dx} [/tex]

[tex] \sf \frac{df}{dx} (x) = 2v \cdot - \csc ^2 (u) \cdot 8x^3 [/tex]

[tex] \sf \frac{df}{dx} (x) = 2 \cot(2x^4) \cdot - \csc ^2 (2x^4) \cdot 8x^3 [/tex]

[tex] \sf \frac{df}{dx} (x) = -16x^3 \cot(2x^4) \csc ^2 (2x^4) [/tex]
