12. matematika kelas X Pertidaksamaan irasional tolong di jawab ​


Penjelasan dengan langkah-langkah:

√(x² - 4) < √(x + 8)

syarat bentuk akar agar terdefenisi, bentuk akar haruslah positif atau nol, hingga

x² - 4 ≥ 0 atau x + 8 ≥ 0

(x + 2)(x - 2) ≥ 0 atau x ≥ -8

x ≥ 2 atau x ≤ -2 atau x ≥ -8

Irisan syarat keterdefenisian

-8 ≤ x ≤ -2 atau x ≥ 2

Kembali ke soal

√(x² - 4) < √(x + 8)

[√(x² - 4)]² < [√(x + 8)]²

x² - 4 < x + 8

x² - x - 4 - 8 < 0

x² - x - 12 < 0

(x - 4)(x + 3) < 0

-3 < x < 4

Kesimpulan Penyelesaian Pertidaksamaan,

= -8 ≤ x ≤ -2 atau x ≥ 2 atau -3 < x < 4

= -3 < x ≤ -2 atau 2 ≤ x < 4
