Perhitungan nilai lomba tersebut menggunakan aturan berikut : Jika jawaban benar, maka nilainya 4 Jika jawaban salah, maka nilainya -2 Jika tidak dijawab, maka nilainya 0 Pada saat lomba, Gendhis mampu mengerjakan 42 soal, dan 8 soal diantaranya salah. Berapa nilai yang didapat Gendhis saat mengikuti lomba Matematika MGMP ?​


Diketahui : - mengerjakan 42 soal

- salah 8

- benar 34 ( 42 - 8 )

- tidak mengerjakan 8 soal ( 50-42 )

Ditanya : Berapa nilai yg didapat gendhis saat mengikuti lomba matematika MGMP?

Dijawab : benar : 34 × 4 = 136 point

salah : 8 × (-2) = -16 point

tdk d jwb : 8 × 0 = 0 pint

Jumlah:

136 point - (- 16 point ) + 0 point

= 120 point

Jadi point yg di dapat gendhis saat mengikuti lomba matematika MGMP adalah 120 point
