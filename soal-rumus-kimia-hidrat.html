Jawaban:

Rumus senyawa hidrat: BaCl2.2H2O

Penjelasan:

BaCl2.xH2O => BaCl2 + x H2O

massa BaCl2 = n x Mr

4,16 = n x 208

n = 0,02 mol

massa H2O = massa senyawa - massa anhidrat

=> 4,88 - 4,16

=> 0,72 gram

massa H2O = n x Mr

0,72 = n x 18

n = 0,04 mol

Perbandingan mol BaCl2 : H2O = 0,02 : 0,04

=> 1 : 2

Rumus senyawa hidrat: BaCl2.2H2O

Senyawa Hidrat

Senyawa yang mengandung air kristal disebut senyawa hidrat, sedangkan senyawa yang tidak mengandung air kristal disebut senyawa anhidrat. Air kristal merupakan molekul air yang terkandung dalam susunan kristal senyawa.

16. Jika hidrat tembaga(II) sulfat dipanaskan maka beratnya berkurang sebanyak 36%. Rumus molekul hidrat tersebut adalah


Rumus dari tembaga(II) sulfat hidrat tersebut adalah CuSO₄.5H₂O. Rumus tersebut dapat dihitung menggunakan perbandingan mol. Perbandingan tersebut dapat ditentukan dari mol tembaga(II) sulfat dibanding mol air.

Pembahasan

Pada soal di atas diketahui beberapa data yaitu:

Senyawa = CuSO₄.xH₂O  

Massa H₂O = 36 %

Mr Cu = 63,5, S = 32, O = 16, H =1

Adapun yang ditanyakan adalah rumus dari senyawa tembaga(II) sulfat hidrat.

Langkah awal yang perlu kalian lakukan adalah menentukan mol masing-masing senyawa hasil reaksi menggunakan data massa dan Mr zat. Mr dapat ditentukan menggunakan data Ar. Adapun massanya dapat ditentukan dari % massa yang diketahui. Pada soal diketahui bahwa setelah dipanaskan massa senyawa hidrat berkurang 36 % artinya dalam senyawa tersebut terdapat 36 % massa H₂O. Jika massa senyawa dimisalkan 100 gram maka:

Massa H₂O = 36 % x 100 gram = 36 gram

Massa CuSO₄= 100 gram – 36 gram = 64 gram

Selanjutnya mol masing-masing zat dapat ditentukan sebagai berikut:

Menghitung mol CuSO₄

Data yang dapat digunakan adalah data massa CuSO₄ dan data Mr yang dapat ditentukan menggunakan data Ar yang diketahui. Berikut penentuan mol CuSO₄.  

Massa CuSO₄= 64 gram

Mr CuSO₄= Ar Cu + Ar S + 4(Ar O)

                  = 63,5 + 32 + 4(16)

                  = 159,5 gram/mol

Pelajari lebih lanjut tentang Mr di: https://brainly.co.id/tugas/20779161.  

Mol CuSO₄ dapat dihitung menggunakan rumus

Mol CuSO₄ = massa / Mr

                    = 64 / 159,5

                    = 0,4 mol

Menghitung mol H₂O

Data yang dapat digunakan adalah data massa H₂O dan data Mr dari H₂O . Massa H₂O  dapat ditentukan menggunakan data massa total senyawa yaitu 1 gram dan massa CuSO₄ yang diketahui. Berikut tahapan penentuan mol H₂O.

Massa H₂O = 36 gram

Mr H₂O = 2 (Ar H) + Ar O

              = 2 (1) + 16

              = 18 gram/mol

Mol H₂O dapat dihitung menggunakan rumus

Mol H₂O = massa / Mr

               = 36 / 18

               = 2 mol

Menentukan perbandingan mol CuSO₄ dan H₂O

Perbandingan mol CuSO₄ dan H₂O data digunakan sebagai dasar penentuan rumus hidrat dari tembaga(II) sulfat yang diinginkan. Perbandingan mol CuSO₄ dan H₂O dapat ditentukan sebagai berikut.

Mol CuSO₄ : mol H₂O

          0,4   :    2             (dibagi 0,4)

            1     :    5

Pelajari lebih lanjut tentang mol dan perbandingan koefisien di: https://brainly.co.id/tugas/18228152.  

Dengan mensubtitusikan angka perbandingan mol CuSO₄ dan mol H₂O sebagai koefisien reaksi, maka reaksi penguapan hidratnya bisa dituliskan sebagai berikut

CuSO₄.xH₂O → CuSO₄ + xH₂O

Karena mol CuSO₄ : mol H₂O adalah 1 : 5 maka reaksinya menjadi

CuSO₄.5H₂O → CuSO₄ + 5H₂O

Jadi dapat disimpulkan bahwa rumus senyawa hidrat tersebut adalah CuSO₄.5H₂O.

19. Sebanyak 49,9 gram senyawa hidrat dengan rumus CuSO 4XH2O dipanaskan sehingga menghasilkan 31,9 gram CuSO4. a. Tulis reaksi pemanasan hidrat tersebut b.tentukan rumus senyawa hidrat tersebut

massa H2O = 49,9 gram - 31,9 gram = 18 gram
Lalu, cari x dalam xH2O
x = massa / Mr 
x = 18 / 18
x = 1
Jadi rumus senyawa hidrat tsb adalah CuSO4H2O.

massa H2O = 47,8% = 47,8 gr
massa MgSO4 = 100 - 47,8 = 52,2 gr
Mr MgSO4 = 120
Mr H2O = 18

x H2O = mol H2O / mol MgSO4
x H2O = (47,8/18) / (52,2/120)
x H2O = 6

rumus senyawa hidrat = MgSO4. 6H2O
