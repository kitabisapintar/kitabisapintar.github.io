Bangun pada gambar adalah bangun ruang gabungan dari prisma segitiga dan balok.

Rumus luas prisma = (2 x luas alas) +(Keliling alas x tinggi)

Rumus luas balok = 2 ((panjang x lebar)+(panjang x tinggi)+(lebar x tinggi))

Namun perlu diperhatikan bahwa pada gambar ada sisi yang berhimpitan, sisi tersebut tidak termasuk dalam luas bangun secara keseluruhan. Sisi yang berhimpit adalah 1 alas prisma segitiga  dan 1 sisi tegak balok (lebar x tinggi).

Sehingga luasnya menjadi;

Luas prisma = luas alas + (Keliling alas x tinggi)Luas balok = 2 ((panjang x lebar)+(panjang x tinggi)+(lebar x tinggi)) - luas alas prisma

Untuk menentukan keliling alas prisma segitiga, kita cari dulu panjang sisi miring segitiga dengan rumus phytagoras.

[tex]x=\sqrt{(12)^{2} +(16)^{2} } \\x=\sqrt{(144 + 256}\\x=\sqrt{400}\\x=20[/tex]

= luas alas + (Keliling alas x tinggi)

[tex]=(\frac{12.16}{2} )+(48.24)\\=96+1152\\=1248[/tex]

[tex]=(2((18.16)+(18.20)+(16.20)))-(\frac{12.16}{2}) \\=(2((288)+(360)+(320)))-(96) \\=(2.968)-96\\=1936-96\\=1840[/tex]
