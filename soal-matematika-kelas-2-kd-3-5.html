c. y + 3x - 4 = 0

saat x = 0, y = 4saat y = 0, x = 1,33saat x = 1,  y = 1           Gambar dapat dilihat pada gambar 4Kesimpulan

Jawab: y = x - 110

Domain -∞ < x ≤ 1000
Jika x mulai dari 1000, maka:

1000-890 = 110999-889 = 110998-888 = 110997-887 = 110996-886 = 110

Range -∞ < f(x) ≤ 1000
----------------------------------------------------
Penjelasan dengan langkah-langkah:

Misalnya:
bilangan i = x
Bilangan ii = y
x - y = 110, x > y, x ≤ 1000, y ≤ 1000
(ubah ke format persamaan garis)
y = x - 110, y < x, y ≤ 1000, x ≤ 1000

Karena dibatasi x ≤ 1000, maka
Domainnya -∞ < x ≤ 1000

Karena dibatasi y ≤ 1000, maka
Rangenya -∞ < f(x) ≤ 1000

x dan y tak hingga, artinya x dan y bisa apa aja.

--------------------------
Jika -∞ < x ≤ 1000
(x kurang dari dan sama dengan 1000)

Contoh x = 1000
y = 1000-110

Contoh x = 996
y = 996-110
y = 886

------------------------------

(y kurang dari dan sama dengan 1000)

Contoh y = 1000
x - y = 110

x = 110 + 1000
x = 1110

Contoh y = 999
