22. lentukan Nilai x yang memenuhi Persamaan 5-63 13x² +63 2 X-16x 6x 13.5​


Jawaban dan Penjelasan :

Untuk menentukan nilai x yang memenuhi persamaan:

5 - 13x² + 63(2x - 16x) + 6x = 13.5

Kita bisa mulai dengan menyederhanakan persamaan ini:

5 - 13x² + 126x - 1008x + 6x = 13.5

Selanjutnya, kita gabungkan semua suku yang memiliki x:

-13x² + (126x - 1008x + 6x) = 13.5 - 5

-13x² - 876x = 8.5

Sekarang, kita ingin membawa persamaan ini ke bentuk kuadrat, sehingga kita bisa mencari nilai x. Untuk melakukan itu, kita akan menyederhanakan persamaan:

-13x² - 876x = 8.5

Kurangkan 8.5 dari kedua sisi:

-13x² - 876x - 8.5 = 0

Untuk mencari nilai x yang memenuhi persamaan kuadrat ini, kita bisa menggunakan rumus kuadrat:

x = (-b ± √(b² - 4ac)) / (2a)

Dalam persamaan kita, a = -13, b = -876, dan c = -8.5. Mari kita substitusi nilai-nilai ini ke dalam rumus:

x = (-(-876) ± √((-876)² - 4(-13)(-8.5))) / (2(-13))

x = (876 ± √(76716 - 442.5)) / (-26)

x = (876 ± √76273.5) / (-26)

x ≈ (876 ± 276.36) / (-26)

Sekarang kita punya dua kemungkinan solusi:

1. x ≈ (876 + 276.36) / (-26)

2. x ≈ (876 - 276.36) / (-26)

Kalkulasikan kedua solusi ini:

1. x ≈ 32.55 (sekitar)

2. x ≈ 40.41 (sekitar)

Jadi, terdapat dua nilai x yang memenuhi persamaan tersebut, yaitu x ≈ 32.55 dan x ≈ 40.41.

26. 3. JIKA akar² persamaan KUADRAT ADALAH -7dan -9 maka persamANGAN aDALAH .....A . X²-2X-63=0B. X²+2X+63=0C. X² -2X -63 =0 D. X²-16- X+63=0E. X²+16X+63=0 jwb.....​


Jawaban:

Kita dapat menggunakan rumus akar-akar persamaan kuadrat untuk menyelesaikan masalah ini. Jika akar-akar persamaan kuadrat adalah -7 dan -9, maka kita dapat menuliskan persamaan kuadrat dalam bentuk:

(x + 7)(x + 9) = 0

Kita dapat mengalikan faktor-faktor tersebut menggunakan aturan distributif, sehingga:

x² + 16x + 63 = 0

Jadi, persamaan kuadrat yang memiliki akar-akar -7 dan -9 adalah X²+16X+63=0.

Jawaban: E. X²+16X+63=0.

Untuk menemukan persamaan kuadrat dengan akar -7 dan -9, kita dapat menggunakan fakta bahwa jika \(a\) dan \(b\) adalah akar-akar dari persamaan kuadrat \(ax^2 + bx + c = 0\), maka persamaan kuadrat tersebut dapat ditulis sebagai \((x - a)(x - b) = 0\).

Dalam kasus ini, akar-akar \(a\) dan \(b\) adalah -7 dan -9, sehingga persamaan kuadratnya adalah \((x - (-7))(x - (-9)) = (x + 7)(x + 9) = 0\).

Sekarang, mari selesaikan persamaan ini:

\((x + 7)(x + 9) = 0\)

\(x^2 + 9x + 7x + 63 = 0\)

\(x^2 + 16x + 63 = 0\)

Jadi, persamaan kuadrat yang sesuai dengan akar -7 dan -9 adalah \(x^2 + 16x + 63 = 0\). Oleh karena itu, jawaban yang benar adalah **E. X² + 16X + 63 = 0**.

Kita mulai dengan membagi kedua suku di dalam kurung dengan √x², sehingga kita memiliki:

lim x →∞ {√x² x² + x / √x² - √3x / √x² + 1 / √x²}

Jadi, hasil akhirnya adalah tak hingga.
