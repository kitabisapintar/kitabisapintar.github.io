8. matematika kelas VI SD latihan 2 no.1 halaman 40 hitunglah luas segi banyak dibawah ini


Matematika kelas VI SD latihan 2 no.1 halaman 40 hitunglah luas segi banyak dibawah ini
Jawaban
Pendahuluan  

Luas Bangun Datar

1. persegi panjang = panjang x lebar

2. persegi = sisi x sisi

3. segitiga = ½ x alas x tinggi

4. jajar genjang = alas x tinggi

5. trapesium = ½ x jumlah sisi sejajar x tinggi

6. belah ketupat = ½ x diagonal 1 x diagonal 2

7. layang-layang = ½ x diagonal 1 x diagonal 2


Pembahasan  

1. Terdiri dari persegi panjang dan segitiga

Ukuran persegi panjang

Panjang = 57 cm

Lebar = 28 cm

Ukuran segitiga

Alas = (57 – 19 – 20) cm = 18 cm

Tinggi = (51 – 28) cm = 23 cm

Luas segi banyak

= luas persegi panjang + luas segitiga

= (panjang x lebar) + (½ x alas x tinggi)

= (57 x 28) + (½ x 18 x 23)

= 1.596 + 207

= 1.803

= 1.803 cm²


2. Terdiri dari dua buah jajar genjang dengan ukuran

Alas = 65 cm

Tinggi = 31 cm

Luas segi banyak

= 2 x Luas jajar genjang

= 2 x (alas x tinggi)

= 2 x (65 x 31)  

= 2 x 2.015

= 4.030

= 4.030 cm²


Kesimpulan

1. Luas segi banyak = 1.803 cm²

2. Luas jajar genjang = 4.030 cm²

15. Kelas = 5 SD Materi = Operasi Hitung Bilangan Bulat Mapel = Matematika1. 34 + ( -35 × ( -10 )) ÷ 52. 5 × ( 5 × ( -5 )) ÷ 5​


™ MATH GLADIATOЯ

1.) 34 + ( -35 × ( -10 )) ÷ 5

= 34 + ( 35 × 10 ) ÷ 5

= 34 + ( 350 ÷ 5 )

= 34 + 70

= 104

2.) 5 × ( 5 × ( -5 )) ÷ 5

= 5 × - (5 × 5) ÷ 5

= 5 × (( -25 ) ÷ 5 )

= 5 × ( -5 )

= -25

☆Jawaban :

》Nomor (1):

34 + (-35 × (-10)) ÷ 5 = ...?

Ingat sifat bilangan bulat yaitu :

(-a) × (-b) = ab

Maka, kita selesaikan kembali soalnya :

= 34 + (35 × 10) ÷ 5

= 34 + 350 ÷ 5

Utamakan pembagian karna lebih kuat.

= 34 + 70

= 104

[tex]\:[/tex]

》Nomor (2):

5 × (5 × (-5)) ÷ 5 = ...?

Ingat sifat bilangan bulat yaitu :

a × (-b) = -(ab)

Maka, kita selesaikan kembali soalnya :

= 5 × (-25) ÷ 5

= (-125) ÷ 5

= -25
