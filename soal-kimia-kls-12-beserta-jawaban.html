7. Kimia kls 12 reaksi biloks


Penjelasan:

1.

biloks F dalam F2 → 0

karena F2 adalah unsur bebas/ tidak berikatan dg unsur lain

2.

Biloks Br pada HBr adalah -1.

Diperoleh dari subtitusi biloks yang sudah diketahui pada HBr untuk memperoleh biloks Br

Biloks total HBr adalah 0 ( aturan biloks 2) , dan biloks H adalah +1 ( Aturan biloks no 8)

HBr = 0

(jumlah atom H x biloks atom h) + (jumlah atom Br x biloks atom Br)

( 1x 1 ) + ( 1x Biloks Br ) = 0

1 + Biloks Br = 0

Biloks Br = 0 - 1

Biloks Br = -1

3.

Mg(s) = Mg^2+(s) + 2e

Biloks Mg adalah 0, sedangkan biloks Mg^2+ adalah +2 sehingga disebut dengan reaksi oksidasi.

5.

FeSO4

Biloks Fe + biloks S + 4.biloks O = 0

(+2) + biloks S + 4. (-2) = 0

(+2) + biloks S + (-8) = 0

Biloks S + (-6) = 0

Biloks S = +6

Biloks Fe = +2

Biloks O = -2

6.

K2 + Cr2+ O7 = 0

+2 + Cr2+ -14 = 0

Cr2 + -12 = 0

Cr2 = +12

Cr = +12 / 2 = +6

Jadi Biloks Cr dalam K2Cr2O7 adalah +6.

7.

biloks Al=3+ , biloks Cl (-) atau -1. jadi reaksinya

AlCl3⇒Al³⁺ + 3Cl⁻

8.

Biloks dasar K = +1

Biloks I pada KI maupun CuI = -1

NaClO4

(1x(+1))+(1xCl))+(4x(-2))=0

Tentukan Mr dari:

a. Fe₂O₃

b. MgSO₄

c. Na₂CO₃

d. C₃H₈O

——————————————

Mr = ∑ Ar = ∑ (Angka Indeks × Ar)

contoh : HₘIₙ ➡ m dan n adalah angka indeks

a ][

Fe₂O₃

Mr = ∑ Ar

Mr = 2 × Ar Fe + 3 × Ar O

Mr = 2 × 56 + 3 × 16

Mr = 160

b ][

MgSO₄

Mr = ∑ Ar

Mr = 1 × Ar Mg + 1 × Ar S + 4 × Ar O

Mr = 1 × 24 + 1 × 32 + 4 × 16

Mr = 120

c ][

Na₂CO₃

Mr = ∑ Ar

Mr = 2 × Ar Na + 1 × Ar C + 3 × Ar O

Mr = 2 × 23 + 1 × 12 + 3 × 16

Mr = 106

d ][

C₃H₈O

Mr = ∑ Ar

Mr = 3 × Ar C + 8 × Ar H + 1 × Ar O

Mr = 3 × 12 + 8 × 1 + 1 × 16

Mr = 60

33. Tolong Ya kak, Soal Kimia kls 11 ​


Jawaban:

1.pH NaOH = 11+log5

2.pH Ca(OH)2 = 12+log5

penjelasan ada pada gambar⬆️⬆️⬆️

Jawaban:

mol NaOH

= massa / Mr

= 0,4 / 40

= 0,01

Molaritas NaOH

= mol / volume

= 0,01 / 2 L

= 0,005 M

NaOH ---> Na+ + OH-

b = 1

Konsentrasi OH-

= M x b

= 0,005 x 1

= 5 x 10^(-3)

pOH

= -log (5 x 10^(-3))

= -log 5 + 3

= 3 -log 5

pH = 14 - pOH = 14 -(3-log 5 ) = 11 + log 5

mol Ca(OH)2

= 0,74 / 74

= 0,01

Molaritas Ca(OH)2

= 0,01 / 0,4 L

= 0,025 M

Ca(OH)2 ---> Ca2+ + 2OH-

b = 2

Konsentrasi OH-

= M x b

= 0,025 x 2

= 0,05

= 5 x 10^(-2)

pOH

= -log (5 x 10^(-2))

= -log 5 + 2

= 2 -log 5

pH = 14 - pOH = 14 -(2-log 5 ) = 12 + log 5
