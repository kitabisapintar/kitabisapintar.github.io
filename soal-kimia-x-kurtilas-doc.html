14. Hitung nilai x dan besar sudut DOC ​


∠COD  = 54°

PembahasanDiketahui :

Perhatikan gambar!

∠AOC = 90°

∠AOC = 36°

Ditanyakan :

∠DOC = 90°

Jawab :

Pada gambar ∠AOB = 180°

maka ∠AOC + ∠COD + ∠DOB  = 180°

⇔ ∠AOC + ∠COD + ∠DOB  = 180°

⇔    90°   + ∠COD +   36 °   = 180°

⇔              ∠COD  +   126 °   = 180°

⇔                              ∠COD  = 180° - 126°

⇔                              ∠COD  = 54°

∴ Jadi besar ∠COD  = 54°
