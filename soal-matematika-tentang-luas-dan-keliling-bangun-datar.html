Luas ( L ). Keliling ( K )

Trapesium

L = ½ × ( A + B ) × t

K = AB + BC + CD + DA

jajaran genjang

L = Alas × Tinggi

K = 2 × ( A + B )

belah ketupat

L = ½ × d1 × d2

K = 4 × S ( sisi )

layang"

L = ½ × d1 × d2

K = 2 × ( A + C )

lingkaran

L = π × r × r

K = 2 × π × r

semoga bermanfaat

1. Trapesium

Rumus Luas =

[tex] \frac{1}{2} (a + b) \times l[/tex]

Rumus Keliling =

[tex]a + b + c + d[/tex]

2. Jajaran Genjang

Rumus Luas =

[tex]alas \times tinggi[/tex]

Rumus Keliling =

[tex]2(a + b)[/tex]

3. Belah Ketupat

Rumus Luas =

[tex] \frac{1}{2} \times {d}^{1} \times {d}^{2} [/tex]

Rumus Keliling =

[tex]4 \times s[/tex]

4. Layang-Layang

Rumus Luas =

[tex] \frac{1}{2} \times {d}^{1} \times {d}^{2} [/tex]

Rumus Keliling =

[tex]2 (a + c)[/tex]

5. Lingkaran

Rumus Luas =

[tex]\pi \times r \times r[/tex]

Rumus Keliling =

[tex]2 \times \pi \times r[/tex]
