Nilai dari 2² + 2³ + 2⁴ = 4 + 8 + 16 = 28Nilai dari 2² × 2³ × 2⁴ = 4 × 8 × 16 = 512Nilai dari 5³ + 5⁻³ = 125 + 1/125 = [tex]125\frac{1}{125}[/tex]Hasil dari 4⁻² +4⁻³ = 1/16 + 1/64 = [tex]\frac{80}{1024}[/tex]2⁻² + 3⁻³ + 1⁻⁴ =  1/4 + 1/27 + 1 = [tex]1\frac{31}{108}[/tex]

Pembahasan

Nilai dari 2² + 2³ + 2⁴ =

2² = 2 × 2  = 42³ = 2 × 2 × 2 = 82⁴ = 2 × 2 × 2 × 2 = 16

Jadi 4 + 8 + 16 = 28

Nilai dari 2² × 2³ × 2⁴ =

2² = 2 × 2  = 42³ = 2 × 2 × 2 = 82⁴ = 2 × 2 × 2 × 2 = 16

Jadi 4 × 8 × 16 = 512

Nilai dari 5³ + 5⁻³ =

5³ = 5 × 5 × 5 = 1255⁻³ = [tex]\frac{1}{5^{3} }[/tex] = [tex]\frac{1}{5}[/tex] × [tex]\frac{1}{5}[/tex] × [tex]\frac{1}{5}[/tex] = [tex]\frac{1}{125}[/tex]

Jadi 125 + [tex]\frac{1}{125}[/tex] =125[tex]\frac{1}{125}[/tex]

Hasil dari 4⁻² +4⁻³ =

4⁻² = [tex]\frac{1}{4^{2} }[/tex] = [tex]\frac{1}{4} } x \frac{1}{4 } x \frac{1}{4} } =[/tex] [tex]\frac{1}{16}[/tex]4⁻³ = [tex]\frac{1}{4^{3} }[/tex] = [tex]\frac{1}{4} x \frac{1}{4^{3} } x \frac{1}{4^{3} } =[/tex] [tex]\frac{1}{64}[/tex]

Jadi [tex]\frac{1}{16} + \frac{1}{64} = \frac{80}{1024}[/tex] = [tex]\frac{5}{64}[/tex]

2⁻² + 3⁻³ + 1⁻⁴ =  

2⁻² = [tex]\frac{1}{2^{2} }[/tex] = [tex]\frac{1}{2} x \frac{1}{2 } = \frac{1}{4}[/tex] 3⁻³ = [tex]\frac{1}{3^{3} } = \frac{1}{3} x \frac{1}{3} x \frac{1}{3} = \frac{1}{27}[/tex] 1⁻⁴ = 1

Jadi [tex]\frac{1}{4} + \frac{1}{27} + 1 = 1\frac{31}{108}[/tex]

Jawab:

Berikut adalah jumlah permutasi untuk setiap himpunan bilangan:

sebuah. Ada 9 angka total, 2 di antaranya 2s, 3 di antaranya 3s, dan 4 di antaranya 2s. Banyaknya permutasi dari himpunan bilangan tersebut adalah :

9! / (2! * 3! * 4!) = 9! / (2 * 3 * 4) = 9 * 8 * 7 = 504

b. Ada 8 angka total, 3 di antaranya 2s, 3 di antaranya 3s, 2 di antaranya 4s, dan 0 di antaranya 5s. Banyaknya permutasi dari himpunan bilangan tersebut adalah :

8! / (3! * 3! * 2!) = 8! / (3 * 3 * 2) = 8 * 7 * 6 = 336

c. Ada 8 angka total, 2 di antaranya 2s, 2 di antaranya 3s, 2 di antaranya 4s, dan 2 di antaranya 5s. Banyaknya permutasi dari himpunan bilangan tersebut adalah :

8! / (2! * 2! * 2! * 2!) = 8! / (2 * 2 * 2 * 2) = 8 * 7 * 6 * 5 = 1.680

d. Ada 7 bilangan bulat, 4 diantaranya 4s, 2 diantaranya 5s, dan 1 diantaranya 7. Banyaknya permutasi dari himpunan bilangan tersebut adalah:

7! / (4! * 2! * 1!) = 7! / (4 * 2 * 1) = 7 * 6 * 5 = 210

Perlu diingat bahwa permutasi adalah susunan sekumpulan item dalam urutan tertentu, sedangkan kombinasi adalah pemilihan item tanpa memperhatikan urutan pemilihannya. Rumus untuk menghitung jumlah permutasi dan kombinasi sedikit berbeda.

Dalam menghitung operasi penjumlahan dan pengurangan bilangan pecahan, maka langkah langkah yang harus dilakukan adalah :

Menyederhanakan bentuk pecahan campuran menjadi pecahan biasa.Apabila penyebut pecahan tersebut berbeda, maka kita harus mencari KPK dari penyebut tersebut.Penyelesaian Soal :

1. ²/₄ + 2³/₄ = ²/₄ + ¹¹/₄

                  = ¹³/₄

                  = 3¹/₄

2.  2¹/₆ + ²/₃ = ¹³/₆ + ²/₃

                   = ¹³/₆ + ⁴/₆

                   = ¹⁷/₆

                   = 2⁵/₆

3. 3¹/₃ + 2¹/₃ = ¹⁰/₃ + ⁷/₃

                    = ¹⁷/₃

                    = 5²/₃

4. 2¹/₂ + 4¹/₄ = ⁵/₂ + ¹⁷/₄

                    = ¹⁰/₄ + ¹⁷/₄

                    = ²⁷/₄

                    = 6³/₄

5. 3²/₃ + 6³/₄ = ¹¹/₃ + ²⁷/₄

                     = ⁴⁴/₁₂ + ⁸¹/₁₂

                     = ¹²⁵/₁₂

                     = 10⁵/₁₂
