4. Menggunakan tabel

[tex]aN_2+bH_2- > c NH_3[/tex]

atom        kiri         kanan        

  N           2a             c

  H           2b            3c

didapat:

2a = c

anggap a = 1, maka c = 2

2b = 3c

2b = 6

b = 3

reaksinya menjadi:

[tex]N_2+3H_2- > 2NH_3[/tex]    

5.

[tex]aC_2H_5OH+bO_2- > cCO_2+dH_2O[/tex]

atom         kiri          kanan

  C             2a              c

  H             6a             2d

  O           a+2b         2c+d

didapat:

2a = c

anggap a = 1, maka c = 2

6a = 2d

6 = 2d

3 = d

a+2b = 2c+d

1+2b = 4+3

1+2b = 7

2b = 6

b = 3

reaksinya menjadi:

[tex]C_2H_5OH+3O_2- > 2CO_2+3H_2O[/tex]

6.

[tex]SO_2+O_2- > SO_3[/tex]

beri huruf didepan masing-masing senyawa:

[tex]aSO_2+bO_2- > cSO_3[/tex]

menggunakan tabel:

atom      kiri        kanan

  S           a             c

  O       2a+2b       3c

didapat:

a = c

anggap a = 1, maka c = 1

2a+2b = 3c

2+2b = 3

2b = 1

b = 1/2

reaksinya menjadi:

[tex]SO_2+\frac{1}{2} O_2- > SO_3[/tex]

hilangkan pecahan dengan cara mengalikan 2 kedua ruas:

[tex]2SO_2+O_2- > 2SO_3[/tex]

reaksi setara.

untuk mencari jumlah volume gas, gunakan perbandingan volume dan koefisien:

[tex]\frac{koefisien SO_3}{koefisien O_2} =\frac{volume SO_3}{volume O_2}[/tex]

[tex]\frac{2}{1}=\frac{0,2}{volumeO_2}[/tex]

[tex]volumeO_2=\frac{0,2}{2}[/tex]

[tex]volumeO_2=0,1L[/tex]

jadi, volume gas oksigen yang bereaksi adalah 0,1 Liter.

24. kak tolong jawab soal kimia ini sama caranyaa yaaa, butuh buat latihan ujian


Jawaban:

D. 4,440 kJ

Penjelasan:

Diket : ΔHf CH₄ = -75 kJ/mol

           ΔHf CO₂ = -393 kJ/mol

           ΔHf H₂O = -285 kJ/mol

           Mr CH₄ = 12 + 4 = 16 gr/mol

Dit : ΔHr 80 gr CH₄ = ?

Jawab :

*tentukan persamaan reaksi  : CH₄ + 2O₂ ⇒ CO₂ + 2H₂O

*tentukan ΔH pembakaran per mol CH₄

ΔHr = ΔHf CO₂ + 2.ΔHf H₂O - ΔHf CH₄

       = -393 + 2 (-285) - (-75) = -393 -570 + 75 = -888 kJ/mol

*tentukan mol CH₄ untuk 80 gr CH₄

mol = massa / Mr = 80 /16 =  5 mol

*tentukan ΔH untuk 5 mol CH₄

ΔH = mol x ΔHr = 5 x -888 kJ/mol = -4,440 kJ

(tanda - berarti reaksi eksotermik atau melepaskan panas)

27. Bisa bantu aku nggak soal kimia latihan 1.5


Penamaan senyawa alkuna:

4,5 dimetil 2 heptuna.2 pentuna4,4 dimetil 1 butana1,5,5 trimetil 3 heksuna7 metil 1 heptuna

Pembahasan

Alkuna adalah senyawa hidrokarbon yang namanya mengikuti aturan tata nama senyawa alkuna. Ciri khas senyawa alkuna adalah adanya ikatan rangkap tiga dalam rantai karbon. Jumlah atom karbon dan hidrogen dalam alkuna sesuai dengan rumus umum alkuna CₙH₂ₙ₋₂. Nama senyawa alkuna berupa rantai lurus meliputi:

etuna, propuna, n-butuna, n-pentuna, n-heksuna, n-heptuna, n-oktuna, n-nonuna, dan n- dekuna.

Senyawa alkuna digunakan sebagai bahan baku pembuatan bahan sintetis seperti plastik, proses pengelasan, pemotongan besi dan logam, serta mempercepat proses pematangan buah.

Tata nama senyawa alkuna rantai bercabang mengikuti aturan:

Penentuan rantai C alkuna terpanjang (fokus pada atom C) Tentukan gugus alkil. Beri nomor pada senyawa alkuna, yaitu rantai C terpanjang, dan berikan dupleks nomor terdekat. Jika ada lebih dari satu gugus alkil, tulis  gugus alkil dengan awalan. Jika ada dua gugus alkil, ambil di (di=2, tri=3, tetra=4, penta=5).  Alkil ditulis dalam urutan abjad. Misalnya, ada gugus metil dan etil. Menurut abjad, e datang sebelum m. Kemudian tulis pertama etil, lalu metil, etil-metil. Singkatnya, tulis nama alkuna seperti ini: Posisi Alkil - Nama Alkil - Posisi Rantai- Rantai Utama.
