13. Tipe bilangan bulat dalam bahasa pascal dikenal sebagai ......


Jawaban:

TIPE DATA PADA PROGRAM PASCAL

Setiap data yang ada pada program pascal memiliki tipe data, baik berupa angka bulat( integer ), angka biasa( real ), atau berupa karakter( char ), (String) dan sebagainya.

Jenis - Jenis data yang di kenal dalam Bahasa Pascal yaitu:

1. TIPE DATA SEDERHANA :

a. Bilangan bulat ( integer )

Tipe data ini menerima angka dan bisa dilakukan proses aritmatika. Tipe data ini adalah tipe data bilangan bulat, yaitu Bilangan atau angka yang tidak memiliki titik desimal atau pecahan.

2. Bilangan biasa ( real )

Bilangan atau angka yang bisa memiliki titik desimal atau pecahan, Tipe data ini menerima angka dan koma serta bisa dilakukan proses aritmatika.

3. Boolean

Boolean adalah tipe data yang hanya mempunyai dua kemungkinan, yaitu true dan false. Biasanya digunakan untuk operasi logika dengan ungkapan AND, OR, atau NOT

4. Karakter ( char )

Karakter adalah data tunggal yang mewakili semua huruf, simbol baca, dan juga simbol angka yang tidak dapat dioprasikan secara matematis, misalnya: ’A’, ’B’, .....,’Z’, ’?’,’!’, dst. Tipe data ini adalah tipe data yang menerima kumpulan karakter (huruf, angka, symbol) tipe data ini tidak bisa dilakukan proses aritmatika.

2. String

String adalah tipe data bentukan yang merupakan deretan karakter yang membentuk satu kata atau satu kalimat, yang biasanya diapit oleh dua tanda kutip. Tipe data yang menerima kumpulan karakter (huruf, angka, symbol) tipe data ini tidak bisa dilakukan proses aritmatika

Tipe Data di Definisikan ( User Defined Data Type )

1. Subrange.

Tipe data ini merupakan pemberian batas nilai yang mungkin untuk suatu variabel dari tipe data yang telah di definisikan.

Sebagai contoh, jangkauan yang mungkin dari nilai ujian adalah dari 0 sampai 100.

Type nilai = 0..100;

2. Enumerated

Disebut tipe enumerated (terbilang) karena semua nilai disebut satu persatu

15. pilihan tipe heading yang diinginkan pada tab?​


Jawaban:

Pada sebuah tab, pilihan tipe heading yang diinginkan tergantung pada konteks dan tujuan penggunaan tab tersebut. Berikut adalah beberapa pilihan umum untuk tipe heading pada sebuah tab:

Judul: Tipe heading ini digunakan untuk memberikan nama atau judul pada tab, misalnya “Beranda”, “Profil”, atau “Produk”.Kategori: Jika tab digunakan untuk mengelompokkan konten yang serupa, Anda dapat menggunakan tipe heading berdasarkan kategori, misalnya “Berita”, “Galeri”, atau “Pencarian”.Status: Jika tab digunakan untuk status atau proses tertentu, Anda dapat menggunakan tipe heading yang menggambarkan status atau tahap proses, seperti “Proses”, “Selesai”, atau “Tertunda”.Prioritas: Jika tab digunakan untuk mengatur tugas atau pekerjaan, Anda bisa menggunakan tipe heading berdasarkan prioritas, misalnya “Rendah”, “Sedang”, atau “Tinggi”.Rentalihan: Jika tab digunakan dalam konteks proyek atau pemrograman, Anda dapat menggunakan tipe heading berdasarkan percabangan, seperti “Master”, “Fitur Baru”, atau “Pengembangan”.Tanggal: Jika tab digunakan untuk mengelompokkan dokumen atau konten berdasarkan tanggal, Anda bisa menggunakan tipe heading berdasarkan rentang atau format tanggal, misalnya “Minggu Ini”, “April 2022”, atau “2019-2020”.

Selain itu, terdapat banyak opsi tipe heading yang dapat disesuaikan dengan kebutuhan konteks dan fungsi tertentu dari tab. Penting untuk mempertimbangkan pengalaman pengguna, kejelasan, dan relevansi saat memilih tipe heading pada sebuah tab.

⇒ saat mengubah tipe nomor, penomorannya bisa menyesuaikan. misalnya:
- ketika dipilih I,II,III maka penomoran menjadi angka romawi kapital.
- ketika dipilih i,ii,iii maka penomoran menjadi angka romawi bukan kapital.
- ketika dipilih A,B,C maka penomoran menjadi huruf alfabet kapital.

⇒ saat mengubah posisi (alignment), penomorannya bisa berubah perataan:
- ketika dipilih Left, nomornya berada dalam perataan kiri
- ketika dipilih Right, nomornya berada dalam perataan kanan
- ketika dipilih Centered, nomornya ada dalam perataan tengah.
