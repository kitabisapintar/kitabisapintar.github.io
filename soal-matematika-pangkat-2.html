8. matematika bagaimana cara trik mudah soal (99^4 - 89^4) : (99^2+89^2) keterangan : ^ = pangkat


Trik mudahnya
Ingat bahwa a²-b² = (a - b)(a + b)

Jadi

(99⁴ - 89⁴) : (99² + 89²)
= ((99²)² - (89²)²) : (99² + 89²)
= (99² + 89²)(99² - 89²) : (99² + 89²)
= 99² - 89²
= (99 + 89)(99 - 89)
= 188 x 10
= 1880

= (a⁴ - b⁴) / (a² + b²)

= (a² - b²)
= (a + b)(a - b)
