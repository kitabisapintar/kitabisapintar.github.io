Pembahasan

Pada dialog-dialog di soal, ada beberapa macam expression  yang digunakan di antaranya adalah congratulation dan agreement, berikut adalah penjelasannya lebih lanjut.

Untuk menunjukan kesetujuan atau ketidaksetujuan kita bisa menggunakan expression of agreement and disagreement.

18. matematika Limit Bonus, mohin bantuannya ka ada pemantapan



[tex]lim(x - > 9) \frac{ \sqrt{6 + \sqrt{x} } - 3}{4x - 11 \sqrt{x} - 3 } [/tex]
Untuk menyelesaikan limit ini, pertama kamu kalikan dengan faktor lawan pembilangnya sehingga :
[tex]lim(x - > 9) \frac{ \sqrt{6 + \sqrt{x} } - 3}{4x - 11 \sqrt{x} - 3 } \times \frac{ \sqrt{6 + \sqrt{x} } + 3 }{ \sqrt{6 + \sqrt{x} } + 3 } = lim(x - > 9) \frac{6 + \sqrt{x} - 9 }{(4x - 11 \sqrt{x} - 3)( \sqrt{6 + \sqrt{x} } + 3)} \\ lim(x - > 9) \frac{ \sqrt{x} - 3}{4x - 11 \sqrt{x} - 3 } \times \frac{1}{ \sqrt{6 + \sqrt{9} } + 3} = \frac{1}{6} \times lim(x - > 9) \frac{ (\sqrt{x} - 3)}{ (\sqrt{x} - 3)(4 \sqrt{x} + 1) } \\ \frac{1}{6} \times lim(x - > 9) \frac{1}{4 \sqrt{x} + 1 } = \frac{1}{6} \times \frac{1}{4 \sqrt{9} + 1} = \frac{1}{78} [/tex]
Jadi, jawabannya 1/78.

Jawaban : E.
