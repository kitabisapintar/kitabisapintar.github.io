5. √19+8√3 Yang 19+8√3 nya di dalam akar ya, mohon bantuan nya


√(19+8√3) = √(19 + 2x4√3) = √(19 + 2x√48) = √((16+3) + 2√(16 x 3)) = √16 + √3 = 4 + √3

6. Bentuk sederhana dari √(19+8√3) adalah...


Buat menjadi bentuk: [tex]\sqrt{a+2\sqrt{b}}[/tex], maka:
=> [tex]\sqrt{19+8\sqrt{3}}[/tex]
=> [tex]\sqrt{19+2.4\sqrt{3}}[/tex]
=> [tex]\sqrt{19+2\sqrt{48}}[/tex]
Sekarang, sederhanakan (Apabila dikali hasilnya 48 dan dijumlah hasilnya 19)
Misalkan a x b = 48 dan a + b = 19. Nilai a = 16 dan nilai b = 3 (Nilai ini cocok)
Faktorkan, maka:
=> [tex]\sqrt{19+2\sqrt{48}}[/tex]
=> [tex]\sqrt{(16+3)+2\sqrt{(16)(3)}}[/tex]
=> [tex]\sqrt{16}+\sqrt{3}[/tex]
=> [tex]4+\sqrt{3}[/tex]
