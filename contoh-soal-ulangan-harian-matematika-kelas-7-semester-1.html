21. contoh soal pts matematika kelas 7 semester 1yang bisa jawab aku kasih point lebih​


1. Cindy mempunyai uang sebesar Rp8.000,00 dan Janet mempunyai uang Rp2.000,00 lebihnya dari uang Cindy. Perbandingan uang Cindy dan Janet adalah ….

a. 2 : 8

b. 8 : 2

c. 4 : 5

d. 5 : 4

Jawaban: c

Pembahasan:

Misalkan uang Cindy = C

              uang Janet = J

Maka diperoleh:

C = 8.000

J = C + 2.000 = 8.000 + 2.000 = 10.000

Sehingga,

C : J = 8.000 : 10.000 = 4 : 5

Jadi, perbandingan uang Cindy dan Janet adalah 4 : 5

5. Nilai n yang memenuhi (12 + 8) + (–3n) = –22 adalah…
A. 14 C. –13
B. 13 D. –14

Kunci Jawaban: A
(12 + 8) + (–3n) = –22
20 – 3n = – 22
– 3n = – 22 – 20
– 3n = – 42
n = –3/–42 = 14
6. 72 – (520 : 8) = …
A. 9 C. 7
B. 8 D. 6

Kunci Jawaban: C
72 – (520 : 8) = 72 - 65
= 7

7. Suhu mula-mula suatu ruangan adalah 250° C. Ruangan tersebut akan digunakan untuk menyimpan ikan sehinga suhunya diturunkan menjadi –30° C. Besar perubahan suhu pada ruangan tersebut adalah ....
A. –280° C C. 220° C
B. –220° C D. 280° C

Kunci Jawaban: C
Perubahan suhu = 25°C – (–3°C)
= 25°C + 3°C
= 28°C
