18. Pembahasan materi persamaan kuadrat kelas 10 SMA​


Apabila bentuk umum dari persamaan x2 – 4 = 3(x – 2) merupakan ax2 + bx + c = 0, maka nilai a, b, dan c berturut-turut adalah ….

A. 1, -3, 2

B. 1, -2, 3

C. 1, 3, -2

D. 1, -3, -10

Jawab:

Untuk menentukan nilai a, b, dan c maka kita harus merubah bentuk soal menjadi bentuk umum terlebih dahulu.

Caranya:

⇒ x2 – 4 = 3(x – 2)

⇒ x2 – 4 = 3x – 6

⇒ x2 – 4 – 3x + 6 = 0

⇒ x2 – 3x + 2 = 0

⇒ a = 1, b = -3, dan c = 2

Jawaban: A

20. pertanyaan matematika persamaan kuadrat sma. yg no. 5


agar akar-akarnya berbeda, maka diskriminannya > 0

x² + nx + (n + 3) = 0

a = 1
b = n
c = n + 3

b²- 4ac > 0
(n)² - 4(1)(n + 3) > 0
n² - 4(n + 3) > 0
n² - 4n - 12 > 0
(n - 6)(n + 2) > 0

n = 6 atau n = -2

sehingga nilai n yang memenuhi adalah

n < -2 atau n > 6

karena akar-akarnya tidak positif, maka akar-akarnya negatif, sehingga

n > 0
(n + 3) > 0
n > -3

iris himpunannya

{ (-∞, -2) U (6, ∞) } n (0, ∞) n (-3, ∞)
= { (-∞, -2) U (6, ∞) } n (0, ∞)
= (6, ∞)

jadi, himpunan penyelesaiannya adalah : n > 6
