Ax = A cos α1

Untuk vektor B, maka vektor komponennya:

Bx = B cos α2

Untuk vektor C, maka vektor komponennya:

Cx = C cos α3

Rx = Ax + Bx + Cx

Ry = Ay + By + Cy

R2 = Rx2 + Ry2

atau

R = √(Rx2 + Ry2)
