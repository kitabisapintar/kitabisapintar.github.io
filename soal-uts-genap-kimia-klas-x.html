26. tolong jawab tugas kimia klas 10​


Biloks S dari SO2 adalah 4

Biloks S dari SO3 adalah 6

Biloks Fe dari Fe 2 O3 = 3

Biloks FeO=2

Penjelasan:

Teman teman harus ingat bahwa biloks O pada soal ini adalah -2

Penjelasannya seperti dibawah ini:

●Biloks S dari SO2 adalah 4,caranya Biloks S + 2× Biloks O=0

S + (-4) = 0

S = 4

●Biloks S dari SO3 adalah 6, caranya Biloks S + 3×Biloks O =0

S + (-6) =0

S = 6

■2×Biloks Fe+3×Biloks O=0 (dalam senyawa Fe2O3)

2×Biloks Fe+(-6)=0

2×Biloks Fe =6

Biloks Fe =6÷2

Biloks Fe =3

■Biloks Fe+×Biloks O=0 (dalam senyawa FeO)

Biloks Fe+(-2)=0

Biloks Fe=2

BO Fe .. ?

a. Fe2O3

2 BO Fe - 6 = 0

2 BO Fe = +6

BO Fe = +6/2

BO Fe = +3

BO Fe + (-2) = 0

BO Fe - 2 = 0

BO Fe = +2

Jawaban:

Biloks s dari SO3 = +6

Biloks s dari SO2 = +4

Biloks Fe dari Fe2O3 = +3

Biloks Fe dari FeO = +2

Penjelasan:

SO3

S + (O x 3) = 0

S + (-2 x 3) = 0

S = +6

SO2

S + (O x 2) = 0

S + (-2 x 2) = 0

S = +4

Fe2O3

(Fe x 2) + (O x 3) = 0

(Fe x 2) + (-2 x 3) = 0

(Fe x 2) + (-6) = 0

2Fe = 6

Fe = 6/2

Fe = +3

FeO

Fe + O = 0

Fe + (-2)  = 0  

Fe = +2

Catatan!

Sesuai dengan aturan biloks : Biloks atom O umumnya adalah -2, kecuali jika berikatan dengan atom F, atau dalam senyawaan peroksida dan superoksida. biloks atom O adalah +2 dalam OF₂, dalam senyawa peroksida (misalnya H₂O₂) berharga -1
