15. If You _________ (not behave), I _________ (throw) you out. Type l : Type ll : Type lll : gmn yah ?? soalnya belom di jelasin​


Jawaban:

If You _________ (not behave), I _________ (throw) you out.

Type l : if you don't behave, I will throw you out.

Type ll : if you didn't behave, I would throw you out.

Type lll : if you hadn't behaved, I would have thrown you out.

Penjelasan:

Ini nama nya Conditional sentence atau kalimat pengandaian. Ada 3 tipe.

Tipe 1 pakai present tense

Rumus: If + S + Verb1 + O, S + will + Verb0 + O

Tipe 2 pakai past tense

Rumus: If + S + Verb2 + O, S + would + Verb0 + O

Tipe 3 pakai past perfect tense

Rumus: If + S + had + Verb3 + O, S + would + have + Verb3 + O

Kalimat bisa di balik ya, pakai will dulu baru if.

Note:

* Verb0 maksud nya kata kerja yang tidak ada penambahan atau perubahan apapun.

Kalau masi belum mengerti, bisa tanya guru bahasa Inggris atau googling, ada banyak contoh soal dan penjelasan nya juga.

17. cari 2 soal kalimat di bikin type1 type 2 real type 3 real


Kelas: SMA
Pelajaran: English
Kategori: conditional
Pembahasan: 

Type 1 (possible event)
- If she calls me, I'll ask her.
- If she doesn't call me, I won't call her first.

Type 2 (imagination)
- If the alien attacked earth, I'd try to get to know it.
(the real: alien doesn't attack earth, so I won't try to get to know it)
- if North Korea started the war against USA, there would be a World War III. (the real: North Korea doesn't start the war yet, so there is no WW III.)

Type 3 (regrets)
- I would have sent it earlier, if I had known about it. 
(the real: I didn't know about it, so I wouldn't send it earlier)
- The murderer would have been caught, if the cops hadn't been fooled by him. (the real: the murderer wasn't caught because the cops were fooled by him)

28. 1. Make 2 sentences of type 1( a hope)/ possibility)2. Make 2 sentences of type 2 ( un-real situation at present/future)​


CONDITIONAL SENTENCES

Conditional Type 1

~ If I finish reading the novel, I will go to the market.

~ If my mother comes, I will clean the room.

Conditional type 2

~ If I were you, I would tell the truth to Mr. Rick.

~ If I studied hard, I would get the good mark.

EXPLANATION

Conditional sentence adalah kalimat pengandaian/bersyarat, di mana suatu keinginan dapat terwujud apabila persyaratannya terpenuhi.

Conditional sentence tersusun atas 2 klausa :

~ If clause menggambarkan kondisinya, dan

~ Main clause menggambarkan konsekuensinya.

Conditional tipe 1 [First conditional] menjabarkan keinginan yang kemungkinan terjadi saat ini ataupun nanti-nanti (real possibility) berdasarkan prediksi. [True in the present/future]

Rumus

If+S+verb1(s/es)..., S+will+verb1......

Exp.

If she tells me her problem honestly, I will understand it.

Conditional tipe 2 [Second conditional] menjabarkan keinginan yang kemungkinan kecil terwujud atau diragukan terjadi saat ini atau mendatang. [Untrue in the present/future]Nominal sentence pada conditional tipe 2 menggunakan tobe "were" untuk subyek tunggal dan jamak.

Rumus :

If+S+verb2 ......, S+would+verb1......

Exp.

If I were a President, I would build the huge library
