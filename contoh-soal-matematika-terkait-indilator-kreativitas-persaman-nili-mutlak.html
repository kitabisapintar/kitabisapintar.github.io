10. Nili mutlak dari |x+2| + |3x| + |2x5|=8


Mapel : Matematika
Kelas : X SMA
Materi : Persamaan Nilai Mutlak

Pembahasan :
|x + 2| + |3x| + |2x + 5| = 8

x + 2 + 3x + 2x + 5 = 8
6x + 7 = 8
6x = 1
x = 1/6

x + 2 + 3x + 2x + 5 = -8
6x + 7 = -8
6x = -15
x = -15/6

Hp : {-15/6 , 1/6}
