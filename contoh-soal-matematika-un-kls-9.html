Tentukan sumbu simetri grafik fungsi dibawah ini

[tex]y = {2x}^{2} - 5x[/tex][tex]y = {3x}^{2} + 12x[/tex][tex]y = - 8 {x}^{2} - 16x - 1[/tex]PENJELASAN

MENENTUKAN SUMBU SIMETRI DAN NILAI OPTIMUM

Berikut rumus yang dapat digunakan untuk menentukan sumbu simetri dan titik optimum grafik fungsi

[tex]f(x) = {ax}^{2} + bx + c [/tex]

Rumus sumbu simetri

[tex]x = - \frac{b}{2a} [/tex]

Rumus Nilai optimum

[tex]y = - \frac{d}{2a} = - \frac{ {b}^{2} - 4ac }{2a} [/tex]

JAWAB

[tex]y = {2x}^{2} - 5x [/tex]

[tex]y = {2x}^{2} - 5x \\ \\ a = 2 \\ \\ b = - 5 \\ \\ x = - \frac{b}{2a} \\ \\ \: \: = - \frac{ - 5}{2(2)} \\ \\ \: \: = - \frac{ - 5}{4} \\ \\ \: \: = \frac{5}{4} [/tex]

[tex]y = {3x}^{2} + 12x[/tex]

[tex]y = {3x}^{2} + 12x \\ \\ a = 3 \\ \\ b = 12 \\ \\ x = - \frac{b}{2a} \\ \\ \: \: \: = - \frac{12}{2(3)} \\ \\ \: \: \: = - \frac{12}{6} \\ \\ \: \: \: \: = - 2[/tex]

[tex]y = { - 8x}^{2} - 16x - 1[/tex]

[tex]y = { - 8x}^{2} - 16x - 1 \\ \\ a = - 8 \\ \\ b = - 16 \\ \\ c = - 1 \\ \\ x = - \frac{ b}{2a} \\ \\ \: \: \: \: = - \frac{ - 16}{2( - 8)} \\ \\ \: \: \: \: = - \frac{ - 16}{ - 16} \\ \\ \: \: \: = \frac{16}{ - 16} \\ \\ \: \: \: = - 1[/tex]

40. cuma satu soal,matematika kls 9​


Jawaban:

ditanya :

[tex] {a}^{ \frac{3}{5} } \times {a}^{ - \frac{1}{2} } \div {a}^{ - \frac{2}{5} } [/tex]

jawab :

gunakan rumus

[tex] { a}^{n} \times {a}^{m} = {a}^{n + m} [/tex]

[tex] \frac{ {a}^{n} }{ {a}^{m} } = {a}^{n - m} [/tex]

[tex] {a}^{ \frac{3}{5} } \times {a}^{ - \frac{1}{2} } \div {a}^{ - \frac{2}{5} } = {a}^{ \frac{3}{5} - \frac{1}{2} - ( - \frac{2}{5} } )[/tex]

[tex] {a}^{ \frac{3}{5} + - \frac{1}{2} + \frac{2}{5} } = {a}^{ \frac{6 - 5 + 4}{10} } = {a}^{ \frac{5}{10} } = {a}^{ \frac{1}{2} } [/tex]

[tex] {a}^{ \frac{1}{2} } = \sqrt{a} [/tex]

Maka hasilnya adalah √a
