32. Buatlah contoh soal matriks invers Contoh dan bukan contoh matriks inversDan jelaskan perbedaannya


Jawab:

Pertama-tama kita harus mengetahui ciri-ciri matriks invers

Penjelasan dengan langkah-langkah:

jika matriks A dengan det A ≠ 0, maka matriks A dinamakan matriks non singular (tidak singular).

Setiap matriks non singular selalu memiliki invers.

Contoh :

matriks C

[tex]\\\left[\begin{array}{ccc}5&3\\-7&-4\end{array}\right][/tex]

det C = 5 x (-4) - 3 x (-7) = -20 + 21 = 1

Karena det C = 1 (tidak sama dengan nol), maka matriks C non singular dan memiliki invers.

Jika matriks bukan invers A dengan det A = 0, maka matriks A dinamakan matriks singular (singular).

Contoh : [tex]\left[\begin{array}{ccc}1&2&3\\4&5&6\\7&8&9\end{array}\right][/tex]

Maka akan menghasilkan det = 0

34. Tentukan KONVERS, INVERS, KONTRAPOSISI dari :Implikasi :" Jika siswa kelas XI dapat menjawab soal matematika maka 3 orang akan mendapat hadiah " ​


Konvers, Invers, dan Kontraposisi dari ilmpikasi: "Jika siswa kelas XI dapat menjawab soal matematika maka 3 orang akan mendapat hadiah ", adalah:

Konvers: Jika 3 orang mendapat hadiah, maka siswa kelas XI dapat menjawab soal matematika.Invers: Jika siswa kelas XI tidak dapat menjawab soal matematika, maka tidak ada 3 orang yang mendapat hadiah.Kontraposisi: Jika tidak ada 3 orang yang mendapat hadiah, maka siswa kelas XI tidak dapat menjawab soal matematika.

Penjelasan dengan langkah-langkah:

Konvers dari sebuah implikasi mengubah posisi antara proposisi awal dan akhir sehingga proposisi akhir menjadi proposisi awal. Dalam hal ini, konversi dari "P → Q" menjadi "Q → P" menyatakan bahwa jika "Q" benar, maka "P" juga benar. Invers dari sebuah implikasi mengubah kedua proposisi awal dan akhir menjadi negasi (lawan) dari proposisi asli dan juga mengubah urutan proposisi. Dalam hal ini, inversi dari "P → Q" menjadi "~P → ~Q" menyatakan bahwa jika "P" salah, maka "Q" juga salah. Kontraposisi dari sebuah implikasi adalah inversi dari konversi. Dalam hal ini, kontraposisi dari "P → Q" menjadi "~Q → ~P" menyatakan bahwa jika "Q" salah, maka "P" juga salah.

Diketahui:

Implikasi "Jika siswa kelas XI dapat menjawab soal matematika maka 3 orang akan mendapat hadiah", dapat dituliskan sebagai berikut:

"P → Q", di mana:P = "siswa kelas XI dapat menjawab soal matematika".Q = "3 orang akan mendapat hadiah".

Ditanya:

Konvers = ...?Invers = ...?Kontraposisi = ...?

Jawab:

Konvers dari implikasi ini adalah:

"Q → P", maka: Jika 3 orang mendapat hadiah, maka siswa kelas XI dapat menjawab soal matematika.

Invers dari implikasi tersebut adalah:

"~P → ~Q", maka: Jika siswa kelas XI tidak dapat menjawab soal matematika, maka tidak ada 3 orang yang mendapat hadiah.

Kontraposisi dari implikasi tersebut adalah:

"~Q → ~P", maka: Jika tidak ada 3 orang yang mendapat hadiah, maka siswa kelas XI tidak dapat menjawab soal matematika.
