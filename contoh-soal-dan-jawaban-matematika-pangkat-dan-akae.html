1. hasil dari operasi akae pangkat 3 dari 32768 adalah​

= 32

Penjelasan dengan langkah-langkah:

karena 32 x 32 x 32 =32.768

Hasil dari operasi akar pangkat 3 dari 32.768 adalah 32

Pendahuluan

Bilangan pangkat tiga merupakan bentuk perkalian ulang suatu bilangan sebanyak tiga kali. Misalkan:

1³ = 1 × 1 × 12³ = 2 × 2 × 23³ = 3 × 3 × 34³ = 4 × 4 × 45³ = 5 × 5 × 5

Adapun akar pangkat tiga adalah suatu bilangan yang merupakan kebalikan dari perpangkatan tiga bilangan bulat tersebut. 1, 8, 27, 64, 125, dan seterusnya merupakan bilangan kubik.

Penyelesaian

[tex]{\sf{ \sqrt[3]{32.768} }}[/tex]

Faktorisasi prisma :

32.768 = 2×2×2×2×2×2×2×2×2×2×2×2×2×2×2

maka :

[tex]{\sf{ \sqrt[ 3]{ (2.2.2.2.2.2.2.2.2.2.2.2.2.2.2)} }}[/tex]

[tex]{\sf{ \sqrt[3]{ {2}^{3} \times {2}^{3} \times {2}^{3} \times {2}^{3} \times {2}^{3} } }}[/tex]

[tex]{\sf{ \sqrt[3]{ {(2 \times 2 \times 2 \times 2 \times 2)}^{3} } }}[/tex]

[tex]{\sf{ \sqrt[3]{ {32}^{3} } }}[/tex]

[tex]{\sf{ \sf{ \bold{ { = 32}}} }}[/tex]

Jadi, hasil operasi akar pangkat tersebut adalah 32

11. Diketahui jumlah akae-akar fungsi kuadrat adalah 7/2. jika hasil kali akar-akatnnya adalah 2/3 , persamaan kuadrat tersebut adalah …?


Dari persamaan x1 + x2 = 7/2, kita bisa menuliskan salah satu variabel dalam bentuk persamaan:

x2 = 7/2 - x1

Kemudian substitusikan ke dalam persamaan x1 * x2 = 2/3:

x1 * (7/2 - x1) = 2/3

7x1/2 - x1^2 = 2/3

Kita bisa mengalikan kedua ruas dengan 6 untuk menghilangkan pecahan:

21x1 - 6x1^2 = 4

6x1^2 - 21x1 + 4 = 0

Persamaan kuadrat di atas adalah bentuk umum persamaan kuadrat yang dapat diselesaikan dengan rumus abc.

a = 6, b = -21, c = 4

x1,2 = (-b ± √b^2 - 4ac) / 2a

x1,2 = (21 ± √441 - 96) / 12

x1,2 = (21 ± √345) / 12

Karena kita mencari hasil kali akar-akarnya, maka:

x1 * x2 = [(21 + √345) / 12] * [(21 - √345) / 12] = (441 - 345) / 144 = 2/3

Sehingga persamaan kuadrat yang dicari adalah:

f(x) = 6x^2 - 21x + 4

25. tolong dong bantuannya : akar-akae persamaan kuadrar 2x^2 + 5× + 12 =0 adalah a dan ß. Pers kuadrat yg akarnya 2a+ 4 dan 2ß + 4 adalah...


jumlah akar : α+β = -b/a = -5/2
kali akar : αβ = c/a = 12/2 = 6

jika x1 = 2α+4
      x2 = 2β+4

maka  Pk : x² - (x1+x2) x + x1.x2 = 0
                 x² - (2α+4+2β+4) x + (2α+4)(2β+4) = 0
                 x² - 2(α+β+4) x + 4αβ+ 8(α+β) + 16 = 0
                 x² - 2 (-5/2 +4) x + 4.6 + 8 (-5/2)+ 16 = 0
                 x² + 5x - 8x + 24 - 20+16 = 0
                 x²-3x+20 = 0     

PK :
2x² + 5x + 12 = 0

α + β = -b/a = -5/2
α • β= c/a = 12/2 = 6

Kita cari komposisinya dulu :

Akar-akarnya adalah (2α + 4) dan (2β + 4)

(2α + 4) + (2β + 4) = 2α + 2β + 8
                              = 2(α + β) + 8
                              = 2(-5/2) + 8
                              = -5 + 8
                              = 3

(2α + 4) • (2β + 4) = 2α2β + 8α + 8β + 16
                              = 4 (α • β) + 8 (α + β) + 16
                              = 4(6) + 8(-5/2) + 16
                              = 24 - 20 + 16
                              = 20

Persamaan kuasdrat barunya adalah :
x² - [(2α + 4) + (2β + 4)]x + [(2α + 4) • (2β + 4)] = 0
⇒ x² - 3x + 20 = 0

Jadi, persamaan kuadrat baru yang akar-akarnya (2α + 4) dan (2β + 4) adalah  x² - 3x + 20 = 0
