3. xy' = x sec (y/x) + y

Untuk menyelesaikan PD bentuk ini, kita bisa menggunakan cara substitusi u = y/x, tetapi sebelum itu kita harus mengubah persamaan tersebut menjadi bentuk y/x.

[tex]\begin{aligned}

xy'&=x\sec{\left(\frac{y}{x}\right)}+y\\

\frac{xy'}{x}&=\frac{x\sec{\left(\frac{y}{x}\right)}}{x}+\frac{y}{x}\\

y'&=\sec{\left(\frac{y}{x}\right)}+\frac{y}{x}\end{aligned}[/tex]

Sekarang, misalkan u = y/x maka y = u.x sehingga y' = u+u'x. Substitusikan semua ini ke PD tadi.

[tex]\begin{aligned}

y'&=\sec{\left(\frac{y}{x}\right)}+\frac{y}{x}\\

u+u'x&=\sec{u}+u\\

u'x&=\sec{u}\\

x\frac{du}{dx}&=\sec{u}\\

\frac{dx}{x}&=\frac{du}{\sec{u}}\\

\int{\frac{1}{x}\,dx}&=\int{\frac{1}{\sec{u}}\,du}\\

\ln{x}+c&=\int{\frac{1}{\frac{1}{\cos{u}}}\,du}\\

\ln{x}+c&=\int{\cos{u}\,du}\\

\ln{x}+c&=\sin{u}+d\\

\sin{u}&=\ln{x}+C\\

u&=\arcsin{\left(\ln{xe^C}\right)}\\

y&=x\arcsin{\left(\ln{kx}\right)}\end{aligned}[/tex]

6. [tex]4y^{2}\frac{dy}{dx}-3x=0[/tex]

Lakukan manipulasi aljabar

[tex]4y^{2}\frac{dy}{dx}-3x=0\\4y^{2}\frac{dy}{dx}=3x\\y^{2}\, dy=\frac{3}{4}x\, dx[/tex]

Integralkan kedua ruas

[tex]\int y^{2}\, dy=\int \frac{3}{4}x\, dx\\\frac{1}{3}y^{3}=\frac{3}{8}x^{2}+C\\y^{3}=\frac{9}{8}x^{2}+C[/tex]

Jadi, solusinya adalah [tex]y^{3}=\frac{9}{8}x^{2}+C[/tex]

7. Asumsi soalnya [tex](6x+y^{2})dx+(2xy-3y^{2})dy=0[/tex]

Didapat [tex]M(x,y)=6x+y^{2}[/tex] dan [tex]N(x,y)=2xy-3y^{2}[/tex], sehingga

[tex]\frac{\partial M}{\partial y}=2y\ \ dan \ \frac{\partial N}{\partial x}=2y[/tex]

Karena sama maka merupakan PD eksak

Cari fungsi [tex]f(x,y)=C[/tex], sehingga

[tex]\frac{\partial f}{\partial x}=M\ \ dan\ \ \frac{\partial f}{\partial y}=N[/tex]

Dari hubungan  [tex]\frac{\partial f}{\partial x}=M=6x+y^{2}[/tex], Integralkan terhadap x, diperoleh

[tex]f(x,y)=3x^{2}+xy^{2}+g(y)[/tex]

dimana g(y) konstanta pengintegralan terhadap x

Dari hubungan  [tex]\frac{\partial f}{\partial y}=N[/tex], dan didapat [tex]\frac{\partial f}{\partial y}=2xy+g'(y)=2xy-3y^{2}[/tex]

Jadi, [tex]g'(y)=-3y^{2}[/tex] dan [tex]g(y)=-y^{3}[/tex].

Maka solusinya adalah

[tex]f(x,y)=3x^{2}+xy^{2}-y^{3}=C[/tex]

.

16. Selesaikanlah persamaan differensial ordiner berikut : y'= tan(x + y)


Jawaban:

Persamaan differensial ini adalah persamaan differensial tak tentu. Untuk menyelesaikannya, kita dapat menggunakan metode pemisalan. Mari kita asumsikan \(u = x + y\), sehingga \(du/dx = 1 + dy/dx\).

Kemudian, kita dapat mengganti \(dy/dx\) dalam persamaan dengan \(du/dx - 1\), sehingga kita memiliki:

\[du/dx - 1 = \tan(u)\]

Selanjutnya, kita bisa memisahkan variabel \(u\) dan \(x\):

\[du/dx = 1 + \tan(u)\]

\[du / (1 + \tan(u)) = dx\]

Selanjutnya, kita dapat mengintegrasikan kedua sisi:

\[\int du / (1 + \tan(u)) = \int dx\]

Integrasi pada sisi kiri dapat dilakukan dengan menggunakan substitusi. Substitusi \(v = \tan(u/2)\) dapat digunakan untuk mengintegrasikan \(\int du / (1 + \tan(u))\):

\[2 \int \frac{dv}{1 + 2v} = x + C\]

\[2 \ln|1 + 2v| = x + C\]

\[|1 + 2v| = e^{(x + C)/2}\]

Kemudian, kita kembalikan substitusi \(v = \tan(u/2)\):

\[|1 + 2\tan(u/2)| = e^{(x + C)/2}\]

Terakhir, kita bisa menulis hasilnya sebagai:

\[1 + 2\tan(u/2) = \pm e^{(x + C)/2}\]

Ini adalah solusi umum dari persamaan differensial \(y' = \tan(x + y)\) dalam bentuk implisit.

20. tolong jawab kakak.... selesaikan persamaan differensial dy/dx+2y=e^3x


Materi : Persamaan Diferensial

Diketahui :

PD : dy/dx + 2y = e^3x

Solusinya dengan menggunakan cara PD linear orde 1 karena derajat tertinggi dari orde tertingginya sama dengan 1 dan variabel y berpangkat 1.

ambil p(x) = 2, maka :

£p(x) dx = £ 2 dx = 2x

Jadi, F(x) = e^(£ (p(x)) dx) = e^2x.

Kalikan faktor integrasi ini ke PD, sehingga :

e^2x dy/dx + (2e^2x)y = (e^3x)(e^2x)

e^2x dy/dx + (2e^2x)y = e^5x

Karena, (2e^2x)y adalah turunan pertama dari ye^2x terhadap x, maka :

d/dx (ye^2x) = e^5x

£d/dx (ye^2x) = £e^5x

ye^2x = 1/5 e^5x + C

Kalikan dengan e^(-2x) agar diperoleh y di ruas kiri sehingga :

y = 1/5e^(5x - 2x) + Ce^(-2x)

y = 1/5e^3x + Ce^(-2x)

Jadi, solusinya :
[tex]y = \frac{1}{5} {e}^{3x} + c {e}^{ - 2x} [/tex]

22. Selesaikanlah persamaan differensial ordiner berikut : (2xy dx + dy)ex2 = 0


(2xy dx + dy) * ex^2 = 0

Untuk menyelesaikan persamaan ini, kita dapat menggunakan metode pemisalan variabel. Pertama-tama, kita perhatikan bahwa persamaan ini bisa ditulis ulang sebagai:

2xy dx + dy = 0

Sekarang, kita akan memisahkan variabel x dan y:

2xy dx = -dy

Selanjutnya, kita dapat membagi kedua sisi dengan xy:

2 dx = -dy/y

Selanjutnya, kita akan melakukan integrasi pada kedua sisi persamaan:

∫2 dx = ∫-dy/y

2x = -ln|y| + C

Di sini, C adalah konstanta integrasi.

Untuk menyelesaikan persamaan terakhir ini, kita perlu menyatakan y dalam satu sisi persamaan:

ln|y| = 2x + C

Sekarang, kita dapat menyingkirkan nilai absolut dengan mempertimbangkan dua kemungkinan:

y = e^(2x + C)

y = -e^(2x + C)

Jadi, solusi dari persamaan differensial adalah:

y(x) = e^(2x + C) atau y(x) = -e^(2x + C)

Di mana C adalah konstanta integrasi yang bisa memiliki berbagai nilai tergantung pada kondisi awal yang mungkin diberikan dalam masalah.

28. selesaikan persamaan differensial linear di bawah ini y aksen + ky= e^-kx


y' + ky = e^-(kx)
Ambil P(x) = k, maka faktor integrasinya adalah :
F(x) = e^£P(x) dx = e^£ k dx = e^kx
kalikan F(x) tersebut ke PD, diperoleh :
(e^kx)y' + (ke^kx)y = (e^-(kx))(e^(kx))
(e^kx)y' + (ke^kx)y = 1
(e^kx)dy/dx + (ke^kx)y = 1
Karena (ke^kx)y adalah turunan pertama dari (e^kx)y terhadap x, maka :
[tex] \frac{d}{dx} ( {e}^{kx} y) = 1[/tex]
Integralkan kedua ruasnya sehingga di peroleh :
£d/dx((e^kx)y) = £1
(e^kx)y = x + c (kalikan dengan e^-(kx))
y = xe^-(kx) + ce^-(kx)

29. cari penyelesaian umum dari persamaan differensial dy/dx = -y² x (x² + 2 ) ⁴. kemudian tentukan penyelesaian khususnya untuk nilai y = 6 di x = 0​


MATERI: PERSAMAAN DIFERENSIAL ORDE 1

Jawaban:

[tex]6{(x^2+2)}^5-60y^{-1}-91=0[/tex]

Penjelasan dengan langkah-langkah:

Soal:

[tex]\frac{dy}{dx}=-y^2x{(x^2+2)}^4[/tex]

di x = 0 dan y = 6

Metode penyelesaian PD bentuk tersebut yaitu dengan menggunakan PD terpisah dimana kita mengubah variabel yang berbeda menjadi variabel yang sama.

Pada soal, kita bisa melihat bahwa terdapat variabel y di ruas kanan, maka kita bisa langsung memindahkannya ke ruas kiri.

[tex]\begin{aligned}

\frac{dy}{dx}&=-y^2x{(x^2+2)}^4\\

\frac{1}{y^2}\frac{dy}{dx}&=-x{(x^2+2)}^4\\

y^{-2}\,dy&=-x{(x^2+2)}^4\,dx\end{aligned}[/tex]

Setelah sampai ke tahap ini, selanjutnya tinggal kita integral kan kedua ruas nya, sehingga akan diperoleh:

[tex]\begin{aligned}

\frac{dy}{dx}&=-y^2x{(x^2+2)}^4\\

\frac{1}{y^2}\frac{dy}{dx}&=-x{(x^2+2)}^4\\

y^{-2}\,dy&=-x{(x^2+2)}^4\,dx\\

\int{y^{-2}\,dy}&=-\int{x{(x^2+2)}^4\,dx}\\

-y^{-1}&=-\frac{1}{10}{(x^2+2)}^5+C\,(\times\,(-10))\\

10y^{-1}&={(x^2+2)}^5-10C\\

{(x^2+2)}^5-10y^{-1}-C=0\end{aligned}[/tex]

Karena x = 0 dan y = 6, maka kita substitusi kan saja ke PD yang telah kita selesaikan tadi, sehingga:

[tex]\begin{aligned}

{(0^2+2)}^5-10(6)^{-1}-C&=0\\

2^5-10\left(\frac{1}{6}\right)&=C\\

32-\frac{5}{3}&=C\\

\frac{91}{6}&=C\end{aligned}[/tex]

Jadi, solusi khususnya adalah:

[tex]6{(x^2+2)}^5-60y^{-1}-91=0[/tex]

Semoga membantu.

Solusi dari [tex]\displaystyle{\frac{dy}{dx}+xy-x=0}[/tex] adalah [tex]\displaystyle{\boldsymbol{y=Ce^{-\frac{1}{2}x^2}+1} }[/tex].

PEMBAHASAN

Persamaan diferensial merupakan persamaan yang memuat turunan satu atau beberapa fungsi yang tidak diketahui. Salah satu jenis persamaan diferensial adalah persamaan diferensial variabel terpisah. Pada jenis persamaan diferensial ini variabel x dan variabel y dapat kita pisahkan sehingga solusinya diperoleh dengan mengintegralkan kedua ruas.

[tex]\displaystyle{f(y)dy=f(x)dx}[/tex]

[tex]\displaystyle{\int\limits {f(y)} \, dy =\int\limits {f(x)} \, dx}[/tex]

[tex]\displaystyle{F(y)=F(x)+C~\rightarrow~solusi~PD}[/tex]

.

DIKETAHUI

[tex]\displaystyle{\frac{dy}{dx}+xy-x=0}[/tex]

.

DITANYA

Tentukan solusinya.

.

PENYELESAIAN

[tex]\displaystyle{\frac{dy}{dx}+xy-x=0}[/tex]

[tex]\displaystyle{\frac{dy}{dx}+x(y-1)=0}[/tex]

[tex]\displaystyle{\frac{dy}{dx}=-x(y-1)}[/tex]

[tex]\displaystyle{\frac{dy}{y-1}=-xdx~\to~integralkan~kedua~ruas}[/tex]

[tex]\displaystyle{\int\limits {\frac{dy}{y-1}} \, dy=\int\limits {-x} \, dx }[/tex]

[tex]\displaystyle{ln(y-1)=-\frac{1}{2}x^2+C_1 }[/tex]

[tex]\displaystyle{y-1=e^{-\frac{1}{2}x^2+C_1} }[/tex]

[tex]\displaystyle{y=e^{C_1}e^{-\frac{1}{2}x^2}+1~~~...e^{C_1}=konstanta }[/tex]

[tex]\displaystyle{y=Ce^{-\frac{1}{2}x^2}+1 }[/tex]

.

KESIMPULAN

Solusi dari [tex]\displaystyle{\frac{dy}{dx}+xy-x=0}[/tex] adalah [tex]\displaystyle{\boldsymbol{y=Ce^{-\frac{1}{2}x^2}+1} }[/tex].
