7. soal matematika SMP kelas 1


1.

Dua panjang AB dan CD sejajar. Ini artinya kita bias membuat garis bantu tegak lurus. Seperti di gambar kita bisa membuat segitiga siku-siku dari sudut D ke titik E yang kita buat.

b. Kita akan mencari besar sudut D terlebih dahulu:

A + E + D merah = 180

65 + 90 + D merah = 180

D merah = 180 - 65 - 90

D merah = 25 derajat

D jingga = 90 derajat

Total sudut D = D merah + D jingga = 25 + 90 = 115 derajat.

a. Setelah mendapatkan sudut D, kita bisa mencari sudut B dengan rumus total sudut trapesium.

A + B + C + D = 360

65 + B + 130 + 115 = 360

B = 360 - 65 - 130 - 115

B = 50 derajat

Sudut B = 50 derajat.

2.

Diketahui

a = 4x

b = 3x

t = 8 cm

Luas = 84 cm²

Ditanya:

Panjang a dan b sesungguhnya.

Jawab:

Rumus trapesium = [tex]\frac{(a+b)*t}{2}[/tex]

[tex]84 = \frac{(4x+3x)*8}{2} \\168 = (7x)*8\\168 = 56x\\168 : 56 = x\\3 = x\\\\\text{Panjang a = 4a = 4 x 3 = 12 cm}\\\text{Panjang b = 3a = 3 x 3 = 9 cm}[/tex]

3.

Diketahui:

AB = 18 cm

CD = 20 cm

Luas = 108 cm²

Ditanya:

Keliling trapesium

Jawab:

[tex]108 = \frac{(18+20)*t}{2} \\216 = 38t\\216 : 38 = t\\5,6 = t[/tex]

Taksirkan tingginya ke satuan terdekat, 5,6 ≅ 6

Tinggi = 6 cm

Panjang EC = CD - AB = 20 - 18 = 2 cm

Rumus pythagoras mencari panjang AC

AC = √EC² - AE²

= √2² + 6²

= √40

= 2√10

Keliling trapesium ABCD = CD + DB + BA + AC

= 20 + 6 + 18 + 2√10

= 44 + 2√10

= 44 + 2 * 3,16

= 44 + 6,32

= 50,32 cm

Keliling trapesium ABCD adalah 50,32 cm

Penjelasan dengan langkah-langkah:

**Jawaban Soal 1**

Pada soal 1, kita diminta untuk menggambar grafik dua persamaan berikut pada bidang koordinat Cartesius:

* (1) x + y = 2

* (2) 3x - 2y = 4

**(1) x + y = 2**

Persamaan ini dapat diubah menjadi bentuk y = -x + 2. Dengan demikian, grafik persamaan ini adalah garis lurus dengan gradien -1 dan perpotongan sumbu y 2.

**(2) 3x - 2y = 4**

Persamaan ini dapat diubah menjadi bentuk y = (3/2)x - 2. Dengan demikian, grafik persamaan ini adalah garis lurus dengan gradien 3/2 dan perpotongan sumbu y -2.

Berikut adalah grafik kedua persamaan tersebut pada bidang koordinat Cartesius:

[Image of Grafik persamaan x + y = 2 dan 3x - 2y = 4]

**Penjelasan**

* Garis merah adalah grafik persamaan x + y = 2.

* Garis biru adalah grafik persamaan 3x - 2y = 4.

Kedua garis tersebut berpotongan di titik (2, 2).

**Kesimpulan**

Jadi, jawaban untuk soal 1 adalah sebagai berikut:

* Grafik persamaan x + y = 2 adalah garis lurus dengan gradien -1 dan perpotongan sumbu y 2.

* Grafik persamaan 3x - 2y = 4 adalah garis lurus dengan gradien 3/2 dan perpotongan sumbu y -2.

Kedua garis tersebut berpotongan di titik (2, 2).

Jawaban:

itu jawabannya dah difotoin beserta jalannya, semoga membantu:)

Penjelasan dengan langkah-langkah:

HP = { y - 2x - 3 = 0 }

jalannya tertera di foto

Pencerminan A(x, y)

→ terhadap garis y = -x adalah A'(-y, -x)

→ terhadap garis y = x adalah A'(y, x)

A(x, y) → refleksi y = -x => A'(-y, -x)

A'(-y,-x) → refleksi y = x => A"(-x, -y)

Hingga (x → -x, y → -y)

y = 2x - 3

→ -y = -2x - 3

→ y = 2x + 3 atau 2x - y + 3 = 0

Kesimpulan

Persamaan bayangan garis y = 2x - 3 setelah direfleksikan terhadap garis y = -x, lalu dilanjutkan dengan pengrefleksian terhadap garis y = x adalah y = 2x + 3

Pecahan

Pecahan biasa adalah pecahan yang terdiri dari pembilang ( bilangan terbagi dan letaknya ada di atas ) dan penyebut ( bilangan pembagi dan letaknya ada di bawah ). Bentuk dari pecahan adalah a/b. Menyederhanakan pecahan biasa dapat dilakukan dengan membagi pembilang dan penyebut pecahan dengan bilangan yang sama sampai tidak bisa dibagi lagi.

Mengubah pecahan menjadi desimal

Cara mengubah pecahan menjadi desimal dengan membagi pembilang dengan penyebut.

♑ Penyelesaian soal ♑

Diketahui: 4[tex]\frac{1}{5}[/tex] - 0,253[tex]\frac{4}{8}[/tex] - 1,28,8 - 3[tex]\frac{1}{2}[/tex]

1. 4[tex]\frac{1}{5}[/tex] - 0,25 = ...

= [4 ([tex]\frac{1}{5}[/tex] dijadikan pecahan desimal)] - 0,25

= (4,0 + 0,2) - 0,25

= 4,2 - 0,25

= 3,95

2. 3[tex]\frac{4}{8}[/tex] - 1,2 = ...

= [3 ([tex]\frac{4}{8}[/tex] dijadikan pecahan desimal)] - 1,2

= (3,0 + 0,5) - 1,2

= 3,5 - 1,2

= 2,3

3. 8,8 - 3[tex]\frac{1}{2}[/tex] = ...

= 8,8 - [3 ([tex]\frac{1}{2}[/tex] dijadikan pecahan desimal)]

= 8,8 - (3,0 + 0,5)

Ayu mempunyai 9 kue. Kemudian Siti memberikan 4 kue kepada Ayu. Kemudian, Ayu memakan 3 dari kue tersebut. Berapakah sisa dari kue yang dimiliki oleh Ayu?

Penjelasan dengan langkah-langkah:

Jumlah kue Ayu adalah 9

Siti memberikan kue kepada Ayu sebanyak 4 kue

9+4 = 13

Kemudian, Ayu memakan 3 dari kue tersebut.

13-3 = 10

Jadi, sisa kue Ayu adalah 10 kue.
