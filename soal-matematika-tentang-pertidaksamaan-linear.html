Jawaban:

Untuk menyelesaikan persamaan tersebut, kita akan menjumlahkan kedua suku x pada kedua sisi persamaan dan menjumlahkan kedua konstanta pada kedua sisi persamaan.

3x + 2x = 8 + 4

5x = 12

Kemudian, kita akan membagi kedua sisi persamaan dengan koefisien x.

x = 12/5

Jadi, nilai x yang memenuhi persamaan tersebut adalah 12/5.


2. sebutkan 3 contoh soal beserta jawaban matematika pertidaksamaan linear satu variabel​


Jawaban:

Berikut adalah 3 contoh soal beserta jawaban matematika pertidaksamaan linear satu variabel:

1. Tentukanlah nilai x pada persamaan 3x + 7 < 22.

Jawaban:

3x + 7 < 22

3x < 22 - 7

3x < 15

x < 5

Jadi, nilai x pada persamaan tersebut adalah x < 5.

2. Selesaikanlah pertidaksamaan x/4 - 2 > 3.

Jawaban:

x/4 - 2 > 3

x/4 > 3 + 2

x/4 > 5

x > 5 x 4

x > 20

Jadi, solusi dari pertidaksamaan tersebut adalah x > 20.

Jawaban:

2x - 3 < 9 - x

3x < 12

x < 4

Jadi, interval penyelesaian dari pertidaksamaan tersebut adalah x < 4.
