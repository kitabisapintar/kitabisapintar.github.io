14. tolong dong bkinin soal kimia tentang laju reaksi kelas XI beserta pembahasan nya, soal objektif ya :D


Suatu reaksi dapat berlangsung dua kali lebih cepat setiap kenaikan suhu 10°C. Berapa kali lebih cepatkah reaksi tersebut berlangsung pada suhu 100°C dibanding 20°C?
a. 16 kali
b. 32 kali
c. 64 kali
d. 128 kali
e. 256 kali

Jawaban :

a = 10°C
n = 2
ΔT = 100 - 20 = 80°C

Penyelesaian :

[tex]v_t = n^{ \frac{\Delta T}{a} . \ v_0[/tex]

[tex]v_t = 2^{ \frac{80}{10} . \ v_0[/tex]

[tex]v_t = 2^8 . \ v_0[/tex]

[tex]v_t = 256v_0[/tex]

Jadi, reaksi tersebut berlangsung 256 kali lebih cepat (Jawaban E)

   CH3OH (l)   +  2 O2(g)  ---->  CO2(g)   + 2 H2O( l)

b. ΔH reaksi = ∑ Entalpi produk - ∑ Entalpi pereaksi

   =  (ΔH CO2 + 2 ΔH H2O ) -  ( ΔH CH3OH + 2 ΔH O2  )  

   =  ( - 393,5 + 2 x - 286 ) kj/mol - ( - 238,6 + 0 ) kj/mol

   =  (( - 393,5 - 572 )  + 238,6) kj/mol

   =   - 965,5 + 238,6 kj/mol

   = - 726,9 kj/mol

c. Kalor yang dibebaskan 10 gram metanol

  Mr CH3OH = 32

  mol = gram metanol / Mr metanol = 10 gram / 32 = 0,3125 mol

  maka kalor yang dibebaskan = 0,3125 mol x 726,9 kj/mol = 227,16 kj.

38. tolong bantu jawab soal fisikakelas xi​

E. waktu tempuh gelombang Q sama dengan R

Periude terbesar = Q

cepat rambat gelombang v = λf

untuk P, v = 0,5 . 20 = 10 m/s

untuk R, v = 4. 2,5 = 10 m/s

Waktu tempuh gelombang t = [tex]\frac{n}{f}[/tex]

untuk P, t =20/20 =1 s

untuk Q, t =30/15 = 2s

untuk R, t = 5/2,5 = 2s

jadi Waktu tempuh gelombang Q dan R sama
