20. Kunci jawaban Matematika kelas 9 halaman 31 no 7


Kunci jawaban Matematika kelas 9 halaman 31 no 7

7. Bilangan [tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] setara dengan [tex]2^{y}[/tex] ,untuk y suatu bilangan bulat positif. Nilai y yang memenuhi adalah 2012.

Simak pembahasan berikut.

Pembahasan

Beberapa sifat bilangan berpangkat adalah:

xᵃ . xᵇ = xᵃ⁺ᵇxⁿ = xᵃ⁺ᵇ , dengan a + b = n(xᵃ)ᵇ = xᵃᵇ[tex]\frac{1}{x^{a}}[/tex] = x⁻ᵃ

Sehingga jawaban matematika kelas 9 halaman 31 nomor 7 adalah:

7. [tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] = [tex]2^{y}[/tex]

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] =  [tex]\frac{2^{2013 + 2}+ 2^{2013 + 1}+ 2^{2013}}{14}[/tex]

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] =  [tex]\frac{2^{2} . 2^{2013}+ 2. 2^{2013}+ 2^{2013}}{14}[/tex]

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] =  [tex]\frac{4. 2^{2013}+ 2. 2^{2013}+ 2^{2013}}{14}[/tex]

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] =  [tex]\frac{2^{2013}(4 + 2 + 1)}{14}[/tex]

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] =  [tex]\frac{2^{2013}(7)}{14}[/tex]

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] =  [tex]\frac{2^{2013}}{2}[/tex]

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] = 2²⁰¹³ × 2⁻¹

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] = 2²⁰¹³⁻¹

[tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] = 2²⁰¹²

Ingat! [tex]\frac{2^{2015}+ 2^{2014}+ 2^{2013}}{14}[/tex] = [tex]2^{y}[/tex]

[tex]2^{y}[/tex] = 2²⁰¹²

y = 2012

Jadi nilai y yang memenuhi adalah 2012.

3. Tentukan nilai dari:

a. pⁿ + (-p)ⁿ, untuk p bilangan bulat dan n bilangan asli genap

Karena n bilangan asli genap, maka

(-p)ⁿ = pⁿ

pⁿ + (-p)ⁿ = pⁿ + pⁿ

pⁿ + (-p)ⁿ = 2pⁿ

Jadi untuk p bilangan bulat dan n bilangan asli genap, pⁿ + (-p)ⁿ = 2pⁿ.

b. pⁿ + (-p)ⁿ, untuk p bilangan bulat dan n bilangan asli ganjil

Karena n bilangan asli ganjil, maka

(-p)ⁿ = -(pⁿ)

Sehingga diperoleh

pⁿ + (-p)ⁿ = pⁿ + (-(pⁿ)

pⁿ + (-p)ⁿ = pⁿ - pⁿ

pⁿ + (-p)ⁿ = 0

Jadi untuk p bilangan bulat dan n bilangan asli ganjil, pⁿ + (-p)ⁿ = 0
