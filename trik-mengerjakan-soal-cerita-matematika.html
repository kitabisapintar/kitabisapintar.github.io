11. matematika bagaimana cara trik mudah soal (99^4 - 89^4) : (99^2+89^2) keterangan : ^ = pangkat


Trik mudahnya
Ingat bahwa a²-b² = (a - b)(a + b)

Jadi

(99⁴ - 89⁴) : (99² + 89²)
= ((99²)² - (89²)²) : (99² + 89²)
= (99² + 89²)(99² - 89²) : (99² + 89²)
= 99² - 89²
= (99 + 89)(99 - 89)
= 188 x 10
= 1880

Semoga membantu.... :)




Perlu diingat bahwa :

a² - b² = (a + b)(a - b)
a⁴ - b⁴ = (a² + b²)(a² - b²)

Jadi,

misal 99 = a dan 89 = b
= (a⁴ - b⁴) / (a² + b²)
= (a² + b²)(a² - b²) / (a² + b²)
= (a² - b²)
= (a + b)(a - b)
= (99 + 89)(99 - 89)
= (188)(10)
= 1880
