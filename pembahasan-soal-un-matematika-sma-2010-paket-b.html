26. UN 2009 PAKET A/B, UN 2010 PAKET B Akar-akar persamaan kuadrat x 2 + (a – 1)x + 2 = 0 adalah α dan β. Jika α = 2β dan a > 0 maka nilai a


x² + (a-1)x + 2 = 0 ; akar2 α dan β ; α = 2β

α.β = 2 ; subtitusi α = 2β
2β² = 2
  β² = 1
   β = 1 atau -1

saat β = 1 ; maka α = 2
α+β = -(a-1)
2+1 = -a+1
    a = -2 (ga cocok, karena a > 0)

saat β = -1 ; maka α = -2
α+β = -(a-1)
-2-1 = -a+1
    a = 4 (cocok!)

34. soal ujian terdiri 6 soal paket A dan 4 soal paket B setiap siswa wajib mengerjakan 8 soal yg terdiri dari 5 soal paket A dan 3 soal paket B. jika soal no 1 dan 2 dipaket A wajib dikerjakan, tentukan banyaknya cara siswa mengerjakan soal ujian​


5 soal dari paket A dan nomor 1 adalah wajib. Jadi, siswa hanya perlu memilih 4 soal lagi dari paket A.

3 soal dari paket B dan nomor 1 adalah wajib. Jadi, siswa hanya perlu memilih 2 soal lagi dari paket B.

Sehingga, banyak cara mengerjakan soal yg dapat dilakukan adalah :

5C4 x 3C2 = [5!/(4!1!)] x [3!/(2!1!)]

= 5 x 3

= 15
