3. Tolong Ya kak, Soal Kimia kls 11 ​


Jawaban:

1.pH NaOH = 11+log5

2.pH Ca(OH)2 = 12+log5

penjelasan ada pada gambar⬆️⬆️⬆️

Jawaban:

mol NaOH

= massa / Mr

= 0,4 / 40

= 0,01

Molaritas NaOH

= mol / volume

= 0,01 / 2 L

= 0,005 M

NaOH ---> Na+ + OH-

b = 1

Konsentrasi OH-

= M x b

= 0,005 x 1

= 5 x 10^(-3)

pOH

= -log (5 x 10^(-3))

= -log 5 + 3

= 3 -log 5

pH = 14 - pOH = 14 -(3-log 5 ) = 11 + log 5

mol Ca(OH)2

= 0,74 / 74

= 0,01

Molaritas Ca(OH)2

= 0,01 / 0,4 L

= 0,025 M

Ca(OH)2 ---> Ca2+ + 2OH-

b = 2

Konsentrasi OH-

= M x b

= 0,025 x 2

= 0,05

= 5 x 10^(-2)

pOH

= -log (5 x 10^(-2))

= -log 5 + 2

= 2 -log 5

pH = 14 - pOH = 14 -(2-log 5 ) = 12 + log 5

10. fisika kls 11 materi elastisitas


[ELASTISITAS]
kelas : 10
kata kunci : hukum hooke , konstanta pegas , rangkaian pegas

Diketahui :
k₁ = 200 N/m
k₂ = 400 N/m
m = 0,75 kg
Δx = 5 cm = 0,05 m (dalam SI jadi dibagi 100 sesuai dengan satuan )
anggap percepatan gravitasi (g) = 10 m/s² 
Ditanyakan : k₃ ?
Dijawab:
*Cari dahulu gaya yang bekerja dan konstanta total pegas :
⇒Dalam hal ini gaya yang bekerja adalah gaya berat ,sehingga:
F = w = m x g = 0,75 x 10 = 7,5 N
⇒Cari konstanta total menggunakan hukum Hooke :
F=k.Δx
k = F 
     Δx
k = 7,5 
     0,05
k = 150 N/m

*Mencari konsntanta pegas ke tiga :
⇒Karena k₁ dan k₂ rangkaian paralel didapat :
kp₁ = k₁ + k₂
      = 200 + 400
      = 600 N/m (menjadi satu)
⇒Karena kp₁ dan k₃ rangakaian seri didapat  (ks₁) yang mana ks₁ adalah total konstantanya (k) = 150 N/m,sehingga:
[tex]\mathsf{ \frac{1}{ks_1} = \frac{1}{kp_1} + \frac{1}{k_3} } \\\\ \mathsf{ \frac{1}{150} = \frac{1}{600} + \frac{1}{k_3} } \\\\ \mathsf{ \frac{1}{k_3} = \frac{1}{150} - \frac{1}{600} \rightarrow Samakan~penyebutnya} \\\\ \mathsf{ \frac{1}{k_3} = \frac{600-150}{150~x~600} } \\\\ \mathsf{ \frac{1}{k_3} = \frac{450}{90.000} } \\\\ \mathsf{ \frac{1}{k_3} = \frac{1}{200} } \\\\ \mathsf{k_3=200~N/m}[/tex]

Jadi konstantanya 200 N/m (A).
