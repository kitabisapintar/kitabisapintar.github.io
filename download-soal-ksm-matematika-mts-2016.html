20. SIAPA YG JAGo KSM MTS TOLONG KERJAKAN YA....​


jawaban:

Di ketahui rumus pola tersebut adalah:

[tex]n \times (n + 1)[/tex]

dengan n : tahun-ke

maka [tex]25 \times (25 + 1) = 25 \times 26 = 650[/tex]

Jawaban D.650

Catatan:

Penyelesaiannya di ketahui dengan mencari rumus dari pola.Cari persamaan dari tiap bilangannya,lalu jadikan rumus.

#semoga membantu

pola bilangan

U1 = 1 × 2 = 2

U2 = 2 × 3 = 6

U3 = 3 × 5 = 15

perhatikan pola → 2, 3, 5, ...

a......2.....3.....5.....

b..........1......2....

c..............1.......1.....

2A = c = 1

A = 1/2

3A + B = b

3(1/2) + B = 1

3/2 + B = 1

B = 1 - 3/2

B = -1/2

A + B + C = 2

1/2 - 1/2 + C = 2

C = 2

Un = An² + Bn + C

Un = 1/2 n² - 1/2 n + 2

pola diatas = n × Un = n(1/2 n² - 1/2 n + 2)

Pola = 1/2 n³ - 1/2 n² + 2n

tahun ke 25

= 1/2 (25)³ - 1/2 (25)² + 2(25)

= 15625/2 - 625/2 + 50

= 15000/2 + 50

= 7500 + 50

34. tolong yg jago KSM pliss buat belajar soalnya​


Diketahui :m = banyak huruf idgham bighunnahn = banyak huruf idzhar

Ditanya :Angka satuan dari m²⁰¹⁹ x n¹⁰⁰⁰ = ?

Penyelesaian :

Bagian 1 : Menentukan nilai dari m dan n

Huruf idgham bighunnah adalah ya', nun, mim, dan wawu. Sehingga, banyaknya huruf idgham bighunnah adalah 4.

Jadi, m = 4

Huruf idzhar adalah ghoin, 'ain, kho', kha', ha', dan hamzah. Sehingga, banyaknya huruf idgham bighunnah adalah 6

Jadi, n = 6

Bagian 2 : Menentukan angka satuan tiap angka

4¹ = 4

4² = 16

4³ = 64 -> Angka satuan terulang

...

4²⁰¹⁹ = 2019 / 2 = 1009 sisa 1

Sehingga, angka satuan dari 4²⁰¹⁹ adalah 4

6¹ = 6

6² = 36 -> Angka satuan sama

Karena angka satuan sama, maka angka satuan dari 6¹⁰⁰⁰ adalah 6

Sehingga,

4²⁰¹⁹ x 6¹⁰⁰⁰

= 4 x 6

= 24 (diambil angka 4-nya)

Kesimpulan :

Angka satuan dari m²⁰¹⁹ x n¹⁰⁰⁰ adalah 4 (B)
