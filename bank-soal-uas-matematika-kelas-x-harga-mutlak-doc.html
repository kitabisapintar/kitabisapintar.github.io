30. soal uas MTK minat kelas x


Jawabannya d. -4 atau 8/3

Semoga membantu^^EksPonenSiaL

3^(8x + 11) = 27^(x² + 4x - 7) = 0
3^(8x + 11) = 3^3(x² + 4x - 7)
8x + 11 = 3x² + 12x - 21
3x² + 4x - 32 = 0
(3x - 8)(x + 4) = 0
x = 8/3 atau x = -4

Nilai x yg memenuhi:
-4 atau 8/3
