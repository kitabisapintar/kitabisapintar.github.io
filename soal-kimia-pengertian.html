Untuk mencari massa rata-rata suatu atom kita harus memiliki data terkait isotop-isotop dari atom dari atom yang diinginkan. isotop adalah atom sejenis (memiliki nomor atom yang sama) namun massa atomnya berbeda.

Melalui data yang diketahui kita dapatkan beberapa isotop dari atom X yaitu:

Isotop 1 => kelimpahan = 74, 22 % = 74,22/100 = 0,7422

Massa = 220,9 amu

Isotop 2 => kelimpahan = 12,78 % = 12,78/100 = 0,1278

Massa = 220,0 amu

Isotop 3 => kelimpahan = 13,00 % = 13,00/100 = 0,1300

Massa = 218,1 amu

Untuk mencari massa rata-tara atom X kita dapat menjumlahkan hasil perkalian antara kelimpahan dengan massa dari masing-masing isotop atom tersebut:

Isotop 1 => kelimpahan x massa = 0,7422 x 220,9 =163,952

Isotop 2 => kelimpahan x massa= 0,1278 x 220,0 =28,116

Isotop 3 => kelimpahan x massa = 0,1300 x 218,1 = 28,353

Dari hasil perkalian antara kelimpahan dan massa dari ketiga isotop tersebut dapat dihitung massa rata-rata (Ar) dari atom X adalah:

Ar = isotop 1 + isotop 2 + isotop 3

= 158,534 + 28,116 + 28,353

= 220,421 ≈ 220,4 amu

Jadi melalui perhitungan tersebut dapat disimpulkan bahwa massa rata-rata dari atom X adalah 220,4 amu

Massa atom relatif adalah massa suatu atom yang ditentukan dengan cara membandingkan dengan massa atom standar. Massa atom relatif, disingkat dengan Ar. Berdasarkan IUPAC (International Union of Pure and Applied Chemistry) yang digunakan sebagai standar penentuan massa atom relatif adalah atom C-12. Kemudian didapatkan rumus penentuan massa arom relatif (Ar) adalah perbandingan massa rata – rata unsur dengan 1/12  massa 1 atom C-12.

[tex]\boxed{\boxed{\bold{Ar~X~=~\frac{massa~1~atom}{\frac{1}{12}~x~massa~1~atom~C-12 }}}}[/tex]

dengan:

Ar X = massa atom relatif X

Massa 1 atom C-12 = 12 sma

Massa rata – rata atom X merupakan massa rata – rata dari semua isotop X yang ada di alam. Massa atom relatif tidak mempunyai satuan.

Massa atom suatu unsur dapat juga ditentukan dengan mengguunakan massa isotop dan kelimpahan dari masing-masing isotop yang berada di alam.

Ar X = [tex]\frac{4,037.10^{-23} }{0,166.10^{-23}}[/tex]
