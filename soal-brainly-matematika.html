18. quis brainly soal olimpiade matematika 2015 a²+b²=6ab.berapakah nilai ( a+b) / (a-b) no spam®™.


Ingat aljabar :

(a + b)² = a² + b² + 2ab
(a - b)² = a² + b² - 2ab

maka,

a² + b² = (a + b)² - 2ab
6ab = (a + b)² - 2ab
6ab + 2ab = (a + b)²
8ab = (a + b)²

manipulasi nilai
(a + b)² - 2ab = 6ab - 2ab
(a - b)² = 4ab

maka,

(a + b)² / (a - b)² = 8ab / 4ab
(a + b)² / (a - b)² = 2
(a + b) / (a - b) = ±√2

Jadi, nilainya √2 atau -√2
