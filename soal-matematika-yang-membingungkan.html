11. Bantuin donggg, sy bingung ama soalny ;-; Soal: Buktikan dengan induksi matematika! (soal ad di gmbr) thanks yg udh bantu :)


Jawaban

Terbukti dengan induksi matematika bahwa

[tex] \displaystyle \sum^{\sf{n}}_{\sf{k} = 1} \: \sf \frac{1}{k(k + 1)} = \frac{n}{n + 1} [/tex]

Pendahuluan

Induksi matematika adalah materi yang berkaitan dengan persamaan atau pertidaksamaan suatu variabel secara matematis

Misal ada persamaan :

P(n) = Q(n)

Dan kita diminta untuk membuktikan bahwa persamaan itu benar, maka langkah pertama adalah buktikan bahwa persamaan itu benar untuk n = 1, Jika terbukti bahwa P(1) = Q(1) maka langkah pertama selesai

Kedua, asumsikan bahwa n = k benar, maka n = k + 1 benar, maka P(k + 1) = Q(k + 1) dapat terbukti jika mengambil persamaan yang didapat dari P(k) = Q(k) ataupun sebaliknya karena kita mengasumsikan bahwa n = k benar

Langkah ini juga berlaku untuk pertidaksamaan

Nah, bagaimana jika dalam persamaan atau pertidaksamaan contohnya P(n) ≥ Q(n) ada syarat bahwa n ≥ a atau sebagainya, maka langkah pertama adalah substitusi n = a dan Pertidaksamaan itu akan terbukti jika P(a) ≥ Q(a)

Lalu sama seperti tadi, asumsikan jika n = k benar, maka n = k + 1 benar, supaya P(k) ≥ Q(k) bila k ≥ a kita perlu membuktikan dari P(k + 1) ≥ Q(k + 1) sampai ke persamaan P(k) ≥ Q(k) atau pun sebaliknya

Untuk langkah kedua dan langkah ketiga, kita bisa menggunakan sifat transitif pertidaksamaan yaitu :

1. Jika x < y < z maka x < z

2. Jika x > y > z maka x > z

Diketahui

[tex] \displaystyle \sum^{\sf{n}}_{\sf{k} = 1} \: \sf \frac{1}{k(k + 1)} = \frac{n}{n + 1} [/tex]

Ditanya

Buktikan dengan induksi matematika bahwa pernyataan tersebut benar!

Penyelesaian

[tex] \displaystyle \sum^{\sf{n}}_{\sf{k} = 1} \: \sf \frac{1}{k(k + 1)} = \frac{n}{n + 1} [/tex]

Ubah bentuk notasi sigmanya :

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{n(n + 1)} = \frac{n}{n + 1} [/tex]

Langkah Pertama, kita buktikan bahwa n = 1 benar :

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{n(n + 1)} = \frac{n}{n + 1} [/tex]

[tex] \sf \frac{1}{1(1 + 1)} = \frac{1}{1 + 1} [/tex]

[tex] \sf \frac{1}{2} = \frac{1}{2} [/tex]

Terbukti

Kedua, kita asumsikan bahwa n = k benar.

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{n(n + 1)} = \frac{n}{n + 1} [/tex]

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{k(k + 1)} = \frac{k}{k + 1} [/tex]

Ketiga, yang terakhir, kita buktikan bahwa n = k + 1 itu benar :

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{(k + 1)((k + 1) + 1)} = \frac{k + 1}{(k + 1) + 1} [/tex]

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{(k + 1)(k + 2)} = \frac{k + 1}{k + 2} [/tex]

Ingat, dari suku terakhir, nilai k selalu dikurangi 1 dari suku setelahnya, maka :

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{((k + 1) -1)((k + 2) -1)} + \frac{1}{(k + 1)(k + 2)} = \frac{k + 1}{k + 2} [/tex]

[tex] \sf \underbrace{\sf\tiny\frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{k(k + 1)}}_{\sf \frac{k}{k + 1}} \sf + \frac{1}{(k + 1)(k + 2)} = \frac{k + 1}{k + 2} [/tex]

Ingat, tadi kita baru saja mengasumsikan bahwa

[tex] \sf \frac{1}{1(1 + 1)} + \frac{1}{2(2 + 1)} + \frac{1}{3(3 + 1)} + ..... + \frac{1}{k(k + 1)} = \frac{k}{k + 1} [/tex]

itu benar

maka persamaannya menjadi :

[tex] \sf \frac{k}{k + 1} + \frac{1}{(k + 1)(k + 2)} = \frac{k + 1}{k + 2} [/tex]

[tex] \sf \frac{k(k + 2)}{(k + 1)(k + 2)} + \frac{1}{(k + 1)(k + 2)} = \frac{(k + 1)(k + 1)}{(k + 2)(k + 1)} [/tex]

[tex] \sf \frac{k(k + 2) + 1}{(k + 1)(k + 2)} = \frac{(k + 1)(k + 1)}{(k + 2)(k + 1)} [/tex]

[tex] \sf \frac{k^2 + 2k + 1}{(k + 1)(k + 2)} = \frac{k^2 + 2k + 1}{(k + 2)(k + 1)} [/tex]

Terbukti

Maka, kebenaran

Terbukti benar

Contoh aplikasi deret

(1) Bunga Tunggal (Barisan Aritmatika)

Yaitu metoda pemberian imbalan jasa bunga simpanan yang dihitung berdasarkan modal pokok pinjaman atau modal awal simpanan saja.

Rumus bunga tunggal:

Mn = Mo (1 + in)

Dimana :

Mn = Nilai modal simpanan periode ke-n

Mo = Nilai modal awal simpanan

i = Persentase bunga simpanan

n = Periode pembungaan

(2) Bunga Majemuk (Barisan geometri)

Yaitu metoda pemberian imbalan jasa bunga simpanan yang dihitung berdasarkan besar modal atau simpanan pada periode bunga berjalan

Rumus bunga majemuk:

Mn = Mo (1 + i)n

Dimana :

Mn = Nilai modal simpanan setelah periode ke-n

Mo = Nilai modal awal simpanan

i = Persentase bunga simpanan

n = Periode pembungaan

x² - 5x - 1 = 0

Maka berlaku :

[tex]p \: + \: q \: = \: -\frac{-5}{1} [/tex]

p + q = 5

[tex]p.q \: = \: \frac{-1}{1} [/tex]

p.q = –1

Diketahui, PK baru mempunyai akar-akar persamaan (3p + 1) dan (3q + 1)

Maka, untuk PK baru berlaku :

(3p + 1) + (3q + 1) = (3p + 3q) + 1 + 1

(3p + 1) + (3q + 1) = 3.(p + q) + 2

(3p + 1) + (3q + 1) = 3.(5) + 2

(3p + 1) + (3q + 1) = 15 + 2

(3p + 1) + (3q + 1) = 17

[tex](3p + 1) \: + \: (3q + 1) \: = \: -\frac{b}{a}[/tex]

[tex]-\frac{b}{a} \: = \: 17[/tex]

[tex]-\frac{-17}{1} \: = \: 17[/tex]

=> didapatkan : a = 1 dan b = –17

(3p + 1).(3q + 1) = 9.pq + 3p + 3q + 1

(3p + 1).(3q + 1) = 9.(p.q) + 3.(p + q) + 1

(3p + 1).(3q + 1) = 9.(–1) + 3.(5) + 1

(3p + 1).(3q + 1) = 7

[tex]\frac{7}{1} \: = \: 7[/tex]

=> didapatkan : a = 1 dan c = 7
