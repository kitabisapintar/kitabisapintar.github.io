Penjelasan:

Fe2S3 + 6 O2 => Fe2O3 + 3 SO3

V SO3 ( STP ) = n x 22,4

2,24 = n x 22,4

n = 0,1 mol

Perbandingan koefisien ~ perbandingan mol

SO3 : Fe2S3 = 3 : 1

mol Fe2S3 = ( 1 / 3 ) x 0,1 mol => 0,03 mol

Mr Fe2S3 = 208

massa Fe2S3 = 0,03 mol x 208 => 6,24 gram
