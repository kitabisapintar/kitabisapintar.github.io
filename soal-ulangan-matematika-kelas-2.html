12. suatu tes ulangan matematika kelas VII jika jawaban benar memperoleh skor 3 sedang jawaban yg tidak di isi mendapatkan skor 0 dan Jawaban salah mendapatkan skor 2 dari 40 soal yg ada jono menjawab benar 27 soal yg tidak di isi 3 soal maka skor yg diperoleh jono adalah


Diketahui :
skor benar = 3
Skor salah = 2
skor tidak menjawab = 0
Jumlah soal = 40 soal
Jono menjawab benar = 27
Tidak diisi = 3

Ditanya :
Skor yg diperoleh Jono= ...?

Jawab :
Jumlah skor seluruhnya = 40 x 3 = 120
Skor soal benar yg dijawab Jono = 27x3 = 81
Skor soal salah = (40-27-3) x 2 = 10 x 2 = 20
Skor soal tidak diisi = 3 x 0 = 0

Jadi skor yg diperoleh Jono
= 120 - 81 - 20 - 0 = 19

Jawaban:

n = 10 ; rata-rata = 64

n = 15 ; rata-rata = 63

[tex]rata - rata = \frac{jumlah \: seluruh \: nilai}{n} [/tex]

Jumlah seluruh nilai saat siswa 10 orang = 64 × 10 = 640

Jumlah seluruh nilai saat siswa 15 orang = 63 × 15 = 945

Untuk mencari rata-rata 5 orang tambahan, cari selisih jumlah seluruh nilai = 945 – 640 = 305

[tex]rata - rata \: 5 \: siswa = \frac{305}{5} = 61[/tex]
