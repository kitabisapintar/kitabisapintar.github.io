Jawaban:

B) 45 orang

Penjelasan dengan langkah-langkah:

persentase petani:

= 90° = 25%

persentase pengusaha:

= 100% - (pedagang + buruh + petani)

= 100% - (40% + 20% + 25%)

= 100% - 85%

= 15%

Banyak pengusaha:

= persentase pengusaha / 100% x banyak penduduk

[tex] = \frac{15}{100\ \\ } \times 300[/tex]

[tex] = 15 \times (300 \div 100)[/tex]

[tex] = 15 \times 3[/tex]

= 45 orang

Jawabannya (B)

= (100 - (40 + 20 + 25)%

= (100 - (60 + 25)%

= (100 - 85)%

= 15%

Maka :

= 15%(300)

= (15(300)/100

= (4.500/100)

= 45 Orang

Diagram lingkaran adalah sebuah gambar diagram yang digunakan untuk menunjukkan atau menyajikan sebuah data ataupun hasil angka dalam bentuk lingkaran. Data dalam diagram lingkaran bisa tersaji dalam bentuk angka, derajat, atau persen. Pada soal diatas diminta untuk membuat sebuah diagram lingkaran dari data yang ada. Untuk mengetahui besarnya sudut dari setiap pata digunakan rumus:

Besar sudut = Besarnya data/total data x 360°

Besar sudut buku tulis = 50/120 x 360°

Besar sudut buku tulis = 150°

Besar sudut pensil = 20/120 x 360°

Besar sudut pensil = 60°

Besar sudut bolpoin= 30/120 x 360°

Besar sudut bolpoin= 90°

Besar sudut penggaris = 20/120 x 360°

Besar sudut penggaris = 60°

Untuk menggambarkan diagram tersebut caranya adalah dengan menggambar sebuah lingkaran dan kemudian membaginya menjadi 4 juring dengan besar sudut sesuai besar sudut dari hasil perhitungan diatas dan memberi label sesuai dengan nama bendanya.

Gambar diagram lingkaran dapat dilihat pada lampiran.
