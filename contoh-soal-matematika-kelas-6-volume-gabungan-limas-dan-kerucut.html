17. Tuliskan rumus volume prisma, limas segitiga, kerucut, tabung, bola, dan bangun ruang gabungan serta beri keterangan!​


Jawaban:

. Kubus

Rumus bangun ruang kubus

Volume kubus V = s x s x s

Luas permukaan kubus L = 6 x (s x s)

Keliling kubus K = 12 x s

Luas salah satu sisi L = s x s

2. Balok

Rumus bangun ruang balok

Volume balok V = p x l x t

Luas permukaan balok L = 2 x ( pl + lt + pt)

Diagonal ruang d = √( p2+ l2 + t2 )

Keliling balok K = 4 x (p + l + t)

3. Prisma Segitiga

Prisma segitiga

Volume prisma segitiga V = luas alas x t

Luas permukaan prisma segitiga L = keliling alas x t + 2 x luas alas segitiga

4. Limas Segiempat

Rumus bangun ruang Limas segiempat

Volume limas V = 1/3 x p x l x t

Luas permukaan limas L = luas alas + luas selubung limas

5. Limas Segitiga

Rumus bangun ruang limas segitiga

Volume limas segitiga V = 1/3 x luas alas x t

Luas permukaan L = Luas alas + Luas selubung limas

6. Tabung

Volume tabung V = π x r2 x t

Luas permukaan tabung L = (2 x luas alas) + (keliling alas x tinggi)

7. Kerucut

Volume kerucut V = 1/3 x π x r2 x t

Luas permukaan kerucut L = ( π x r2 ) + ( π x r x s)

8. Bola

Volume bola V = 4/3 x π x r3

Luas permukaan bola L = 4 x π x r2

27. carilah rumus - volume limas- Tinggi limas- Luas alas limas- volume kerucut- tinggi kerucut- luas alas kerucut​


Jawaban:

•volume limas:Rumus volume limas secara umum dinyatakan melalui persamaan luas alas dikali tinggi limas (t). Untuk mencari luas alas limas bergantung pada bentuk alasnya. Pada limas segitiga, luas alas dapat dicari menggunakan rumus L = ½ × a × t, di mana a adalah alas segitiga dan t adalah tinggi segitiga.

•Tinggi limas:Asumsikan tinggi sisi tegak sebagai a dan rusuk alas sebagai s, maka, tinggi limas dihitung melalui teorema phythagoras : t = √[a² - (½s)²].

•Luas alas limas:Pada limas segitiga, luas alas dapat dicari menggunakan rumus L = ½ × a × t, di mana a adalah alas segitiga dan t adalah tinggi segitiga. Sedangkan pada limas segi empat, besar luas alas dapat dicari dengan rumus kuadrat sisi (s2) pada persegi atau hasil kali panjang dan lebar (p × ℓ) pada persegi panjang.

•volume kerucut:jari-jari alas kerucut r = diameter/2 = 84/2 = 42 cm. volume kerucut = (1/3) (22/7) x 42 x 42 x 75 = 138.600 cm3.14

•tinggi kerucut:Tinggi kerucut (t), yaitu jarak dari titik puncak kerucut C ke pusat bidang alas O, yakni ruas garis CO. Selimut kerucut, yaitu sisi kerucut yang tidak diraster yang merupakan bidang lengkung.

•luas alas kerucut:Karena alas permukaan kerucut adalah lingkaran dam selimutnya adalah segitiga maka rumus untuk menghitung luas alas permukaan kerucut adalah luas alas + luas selimut.
