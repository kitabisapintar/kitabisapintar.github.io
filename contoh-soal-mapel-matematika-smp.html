4. KuisMAPEL MatematikaSMP​


✏JAWABAN:

[tex] \frac{1.7}{x} = \frac{4}{9} [/tex]

[tex]\frac{1.7}{x} = \frac{4}{0.09} [/tex]

[tex]x = 1.7 \times 0.09 \div 4[/tex]

[tex]x = 0.153 \div 4[/tex]

[tex]x = 0.03825 \: m[/tex]

[tex]x = 3.825 \: cm[/tex]

Lebar model/miniatur mobil tersebut adalah 3,825 cm

.

Pembahasan

Skala adalah perbandingan antara jarak pada peta dengan jarak sebenarnya.

.

Rumus-rumusSkala

[tex]s = \frac{jp}{js} [/tex]

.

Jarak pada peta

[tex]jp = js \times s[/tex]

.

Jarak sebenarnya

[tex]js = \frac{jp}{s} [/tex]

.

Keterangan:

s=Skala

jp=Jarak pada peta

js=Jarak sebenarnya

.

Skala adalah perbandingan dari jarak pada peta dengan jarak sebenarnya.

.

PenyelesaianDiketahui

Panjang mobil sebenarnya=4 m

Lebar mobil sebenarnya=1,7 m

Panjang model/miniatur mobil=9 cm

.

Ditanya

Lebar model/miniatur mobil tersebut adalah...

.

Jawab

Skala=

[tex]s = \frac{jp}{js} [/tex]

[tex]s = \frac{9 \: cm}{4 \: m} [/tex]

[tex]s = \frac{9 \: cm}{400 \: cm} [/tex]

[tex]s = \frac{9}{400} [/tex]

.

Lebar model/miniatur mobil tersebut=

[tex]jp = js \times s[/tex]

[tex]jp = 1,7 \: m \times \frac{9}{400} [/tex]

[tex]jp = 170 \: cm \times \frac{9}{400} [/tex]

[tex]jp = \frac{170 \times 9}{400} \: cm[/tex]

[tex]jp = \frac{1.530}{400} \: cm[/tex]

[tex]jp = \frac{153}{40} \: cm[/tex]

[tex]jp = 3,825 \: cm[/tex]

.

Jadi, lebar model/miniatur mobil tersebut adalah 3,825 cm

.

14. contoh soal sifat asosiatif dan distributif kelas 5 sd mapel matematika


Kelas : V (5 SD)
Materi : bilangan bulat
Kata Kunci : bilangan, bulat, sifat-sifat, asosiatif, distributif

Pembahasan :
1. Penjumlahan
Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

(a + b) + c = a + (b + c)

contoh :

(2 + 6) + 8 = 2 + (6 + 8)

⇔ 8 + 8 = 2 + 14

⇔ 16 = 16


2. Pengurangan

tidak bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c tidak berlaku:

(a – b) – c ≠ a – (b – c).

contoh :

(3 - 5) - 7 ≠ 3 - (5 - 7)

⇔ -2 - 7 ≠ 3 - (-2)

⇔ -9 ≠ 5


3. Perkalian

Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c, berlaku :

(a x b) x c = a x (b x c)

contoh :

(2 x 5) x 6 = 2 x (5 x 6)

⇔ 10 x 6 = 2 x 30

⇔ 60 = 60.


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

a x (b + c) = (a x b) + (a x c)

contoh : 

–5 x [9 + (–1)] = [–5 x 9] + [–5 x (–1)] 

⇔ –5 x 8 = –45 + 5

⇔ –40 = –40


ii. distributif terhadap pengurangan

a x (b – c) = (a x b) – (a x c)

contoh :

7 x [–10 – 12] = [7 x (–10)] – [7 x 12]

⇔ 7 x -22 = -70 - 84

⇔ -154 = -154.


4. Pembagian

Tidak bersifat asosiatif, sehingga Untuk sembarang bilangan bulat a, b, dan c berlaku:

(a : b) : c ≠ a : (b : c)

contoh :

(8 : 4) : 2 ≠ 8 : (4 : 2)

⇔ 2 : 2 ≠ 8 : 2

⇔ 1 ≠ 4


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

(a + b) : c = (a : c) + (b : c)

contoh : 

(12 + 6) : 2 = (12 : 2) + (6 : 2)

⇔ 18 : 2 = 6 + 3

⇔ 9 = 9


ii. distributif terhadap pengurangan

(a – b) : c = (a : c) – (b : c)

contoh :

(21 – 9) : 3 = (21 : 3) – (9 : 3)

⇔ 12 : 3 = 7 - 3

⇔ 4 = 4

 

Semangat!

c=Konstanta

Sistem persamaan linear satu variabel adalah persamaan yang hanya memiliki satu variabel. Variabel adalah huruf yang digunakan untuk mewakili nilai dari suatu objek yang tidak diketahui nilainya.

.

Bentuk umum dari sistem persamaan linear satu variabel (SPLSV) adalah:

[tex]ax + b = 0[/tex]

.

Keterangan

a=Koefisien

b dan 0=Konstanta

x=Variabel

.

Contoh soal dari SPLSV:

1).

[tex]1 + x = 9[/tex]

[tex]x = 9 - 1[/tex]

[tex]x = 8[/tex]

.

2).

[tex]2 \times y = 4[/tex]

[tex]2y = 4[/tex]

[tex]y = \frac{ 4}{2} [/tex]

[tex]y = 2[/tex]

.

Berdasarkan dari kedua contoh soal di atas kita dapat menyimpulkan, bahwa jika suatu bilangan berpindah ruas, maka operasi bilangan yang diangkatnya akan berubah menjadi kebalikannya.

.

[tex] \sqrt{x + 20} - \sqrt{x + 1} = 1[/tex]

Nilai x adalah...

[tex] \sqrt{x + 20} = 1 + \sqrt{x + 1} [/tex]

[tex] {( \sqrt{x + 20} )}^{2} = {(1 + \sqrt{x + 1} )}^{2} [/tex]

[tex]x + 20 = 1 + 2 \sqrt{x + 1} + x + 1[/tex]

[tex]x + 20 = 1 + 1 + 2 \sqrt{x + 1} + x[/tex]

[tex]x + 20 = 2 + 2 \sqrt{x + 1} + x[/tex]

[tex]x - x + 20 - 2 = 2 \sqrt{x + 1} [/tex]

[tex]18 = 2 \sqrt{x + 1} [/tex]

[tex] \frac{18}{2} = \sqrt{x + 1} [/tex]

[tex]9 = \sqrt{x + 1} [/tex]

[tex] {9}^{2} = {( \sqrt{x + 1} )}^{2} [/tex]

[tex]81 = x + 1[/tex]

[tex]81 - 1 = x[/tex]

[tex]80 = x[/tex]

.
