4. Jika hidrat tembaga(II) sulfat dipanaskan maka beratnya berkurang sebanyak 36%. Rumus molekul hidrat tersebut adalah


Rumus dari tembaga(II) sulfat hidrat tersebut adalah CuSO₄.5H₂O. Rumus tersebut dapat dihitung menggunakan perbandingan mol. Perbandingan tersebut dapat ditentukan dari mol tembaga(II) sulfat dibanding mol air.

Pembahasan

Pada soal di atas diketahui beberapa data yaitu:

Senyawa = CuSO₄.xH₂O  

Massa H₂O = 36 %

Mr Cu = 63,5, S = 32, O = 16, H =1

Adapun yang ditanyakan adalah rumus dari senyawa tembaga(II) sulfat hidrat.

Langkah awal yang perlu kalian lakukan adalah menentukan mol masing-masing senyawa hasil reaksi menggunakan data massa dan Mr zat. Mr dapat ditentukan menggunakan data Ar. Adapun massanya dapat ditentukan dari % massa yang diketahui. Pada soal diketahui bahwa setelah dipanaskan massa senyawa hidrat berkurang 36 % artinya dalam senyawa tersebut terdapat 36 % massa H₂O. Jika massa senyawa dimisalkan 100 gram maka:

Massa H₂O = 36 % x 100 gram = 36 gram

Massa CuSO₄= 100 gram – 36 gram = 64 gram

Selanjutnya mol masing-masing zat dapat ditentukan sebagai berikut:

Menghitung mol CuSO₄

Data yang dapat digunakan adalah data massa CuSO₄ dan data Mr yang dapat ditentukan menggunakan data Ar yang diketahui. Berikut penentuan mol CuSO₄.  

Massa CuSO₄= 64 gram

Mr CuSO₄= Ar Cu + Ar S + 4(Ar O)

                  = 63,5 + 32 + 4(16)

                  = 159,5 gram/mol

Pelajari lebih lanjut tentang Mr di: https://brainly.co.id/tugas/20779161.  

Mol CuSO₄ dapat dihitung menggunakan rumus

Mol CuSO₄ = massa / Mr

                    = 64 / 159,5

                    = 0,4 mol

Menghitung mol H₂O

Data yang dapat digunakan adalah data massa H₂O dan data Mr dari H₂O . Massa H₂O  dapat ditentukan menggunakan data massa total senyawa yaitu 1 gram dan massa CuSO₄ yang diketahui. Berikut tahapan penentuan mol H₂O.

Massa H₂O = 36 gram

Mr H₂O = 2 (Ar H) + Ar O

              = 2 (1) + 16

              = 18 gram/mol

Mol H₂O dapat dihitung menggunakan rumus

Mol H₂O = massa / Mr

               = 36 / 18

               = 2 mol

Menentukan perbandingan mol CuSO₄ dan H₂O

Perbandingan mol CuSO₄ dan H₂O data digunakan sebagai dasar penentuan rumus hidrat dari tembaga(II) sulfat yang diinginkan. Perbandingan mol CuSO₄ dan H₂O dapat ditentukan sebagai berikut.

Mol CuSO₄ : mol H₂O

          0,4   :    2             (dibagi 0,4)

            1     :    5

Pelajari lebih lanjut tentang mol dan perbandingan koefisien di: https://brainly.co.id/tugas/18228152.  

Dengan mensubtitusikan angka perbandingan mol CuSO₄ dan mol H₂O sebagai koefisien reaksi, maka reaksi penguapan hidratnya bisa dituliskan sebagai berikut

CuSO₄.xH₂O → CuSO₄ + xH₂O

Karena mol CuSO₄ : mol H₂O adalah 1 : 5 maka reaksinya menjadi

CuSO₄.5H₂O → CuSO₄ + 5H₂O

Jadi dapat disimpulkan bahwa rumus senyawa hidrat tersebut adalah CuSO₄.5H₂O.

6. Sebuah contoh senyawa MgSO4.X H2O yg beratnya 5,018 gram, dipanaskan sampai semua hidratnya hilang, bobotya 2,445 gram, bgaimana rumus hidratnya


Sebuah contoh senyawa MgSO₄.xH₂O yg beratnya 5,018 gram, dipanaskan sampai semua hidratnya hilang, bobotya 2,445 gram, bgaimana rumus hidratnya

Diketahui:

Massa MgSO₄.xH₂O = 5,018 gram

Massa MgSO₄ = 2,445 gram

Ar Mg =24,31 , S = 32, O = 16, H = 1 (Tabel SPU)

Ditanya: Rumus molekul Hidratnya....?

Jawab:

Berdasarkan uraian soal kita memiliki informasi massa senyawa awal yang mengandung Hidrat (air) dan massa senyawa tanpa kandungan Hidrat (air), maka berdasarkan data tersebut, kita bisa menghitung massa hidrat yang terkandung.

Berdasarkan massa hidrat tersebut kita bisa menghitung mol garam dan mol hidrat, dimana dengan prinsip Stoikiometri yaitu perbandingan mol suatu zat akan sama dengan perbandingan koefisiennya.

maka reaksi senyawa hidrat bisa kita tuliskan sebagai berikut:

MgSO₄.xH₂O(s) ==> MgSO₄(aq) + x H₂O(g)

 5,018 gram            2,445 gram         ?

Massa H₂O (air) = Massa MgSO₄.xH₂O - Massa MgSO₄

Massa air = 5,018 gram - 2,445 gram

Massa air = 2,573 gram

sehingga molnya bisa kita hitung, yaitu:

mol = massa / Mr

Mr MgSO₄ = Ar Mg + Ar S + 4 x Ar O

Mr MgSO₄ = 24,31 + 32 + 4 x 16

Mr MgSO₄ = 56,31 + 64

Mr MgSO₄ = 120,31

selanjutnya kita hitung Mr Air, yaitu:

Mr H₂O = 2 x Ar H + Ar O

Mr H₂O = 2 x 1 + 16

Mr H₂O = 18

maka molnya bisa kita hitung yaitu:

mol MgSO₄ = massa MgSO₄ / Mr

mol MgSO₄ = 2,445 gram / 120,31

mol MgSO₄ = 0,0203 mol

selanjutnya mol air, yaitu:

mol air = massa air / Mr

mol air = 2,573 gram / 18

mol air = 0,1429 mol

maka:

mol MgSO₄ : mol air =  0,0203 mol : 0,1429 mol = 1 : 7

berarti 1 mol MgSO₄ mengikat 7 molekul air, sehingga rumus hidratnya adalah MgSO₄.7H₂O.

12. Sebanyak 49,9 gram senyawa hidrat dengan rumus CuSO 4XH2O dipanaskan sehingga menghasilkan 31,9 gram CuSO4. a. Tulis reaksi pemanasan hidrat tersebut b.tentukan rumus senyawa hidrat tersebut


Jawab 
a. Reaksi pemanasan : CuSO4xH2O ---> CuSO4 + xH2O
                                       49,9 gr            31,9 gr      ....
b. Rumus senyawa hidrat 
Pertama, cari massa dari H2O (gunakan hukum kekekalan massa)
massa H2O = 49,9 gram - 31,9 gram = 18 gram
Lalu, cari x dalam xH2O
x = massa / Mr 
x = 18 / 18
x = 1
Jadi rumus senyawa hidrat tsb adalah CuSO4H2O.

19. sebuah contoh MgSO4 x H2O yang beratnya 5,018 g dipanaskan sampai semua air hidratnya terlepas, hasil senyawa MgSO4 beratnya 2,449 gr bagaimana rumus hidrat itu


massa MgSO4.XH2O : 5,018 gram
massa MgSO4 : 2,449 gram
Rumus Hidrat : ......................... ?

Jawab

Langkah 1
Buat reaksi pemanasan dimana senyawa hidrat akan menghasilkan anhidrat dan air.

Hidrat : MgSO4.XH2O
Anhidrat : MgSO4

Reaksi
MgSO4.XH2O -----> MgSO4 + XH2O

Langkah 2
Tentukan mol MgSO4 dengan rumus :

Mr MgSO4 : 24 + 32 + 16(4)
: 120 g/mol

mol : massa / Mr

mol MgSO4 : m. MgSO4 / Mr MgSO4
: 2,449 g / 120 g/mol
: 0,0204 mol

Langkah 3
Tentukan mol MgSO4.XH2O dengan membandingkan koefisien reaksi terhadap koefisien reaksi MgSO4

mol MgSO4.XH2O : (k. MgSO4.XH2O / k. MgSO4) x mol MgSO4
: (1/1) x 0,0204 mol

Langkah 4
Tentukan nilai Mr MgSO4.XH2O

mol : massa / Mr
Mr : massa / mol

Mr MgSO4.XH2O : 5,018 g / 0,0204 mol
: 245,98 g/mol

Langkah 5
Tentukan nilai XH2O dengan cara menjumlah Mr MgSO4.XH2O lalu bandingkan dengan Mr yg telah diketahui.

Mr MgSO4.XH2O : Mr MgSO4 + Mr X.H2O
245,98 g/mol : 120 g/mol + 18X g/mol
18X g/mol : (245,98 - 120) g/mol
X : 125,98 g/mol / 18 g/mol
X : 6,998
: 7

Jadi, rumus hidratnya adalah MgSO4. 7H2O.

Rumus senyawa hidrat: BaCl2.2H2O

Penjelasan:

BaCl2.xH2O => BaCl2 + x H2O

massa BaCl2 = n x Mr

4,16 = n x 208

n = 0,02 mol

massa H2O = massa senyawa - massa anhidrat

=> 4,88 - 4,16

=> 0,72 gram

massa H2O = n x Mr

0,72 = n x 18

n = 0,04 mol

Perbandingan mol BaCl2 : H2O = 0,02 : 0,04

=> 1 : 2

Rumus senyawa hidrat: BaCl2.2H2O

25. jika hidrat magnesium sulfat dipanaskan maka beratnya berkurang 47,8% rumus senyawa hidrat tersebut adalah


massa H2O = 47,8% = 47,8 gr
massa MgSO4 = 100 - 47,8 = 52,2 gr
Mr MgSO4 = 120
Mr H2O = 18

x H2O = mol H2O / mol MgSO4
x H2O = (47,8/18) / (52,2/120)
x H2O = 6

rumus senyawa hidrat = MgSO4. 6H2O

35. cara menentukan rumus hidrat
